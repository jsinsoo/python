13차시: 파일 입출력(File I/O) 마스터

프로그램의 데이터를 영구적으로 저장하고 불러오는 데이터 관리의 핵심을 배웁니다.

기초

파일 쓰기와 읽기

가장 많이 쓰이는 'w'(쓰기) 모드와 'r'(읽기) 모드를 학습합니다.

💡 with 문을 사용하세요: with 구문을 사용하면 파일을 다 쓴 후 f.close()를 직접 호출하지 않아도 자동으로 닫아주어 안전합니다. 
# 1. 파일 쓰기 ('w' 모드: 덮어쓰기)
with open('hello.txt', 'w', encoding='utf-8') as f:
    f.write("안녕하세요, 파이썬 파일 입출력입니다.\n")
    f.write("두 번째 줄을 작성합니다.")

# 2. 파일 읽기 ('r' 모드)
try:
    with open('hello.txt', 'r', encoding='utf-8') as f:
        content = f.read()
        print(content)
except FileNotFoundError:
    print("파일을 찾을 수 없습니다.")
응용

줄 단위 처리와 추가 모드

대용량 파일을 효율적으로 읽거나 기존 파일에 내용을 덧붙이는 방법을 다룹니다.

2.1 줄 단위로 읽기 (readlines)

with open('hello.txt', 'r', encoding='utf-8') as f:
    for line in f:
        print(f"읽은 줄: {line.strip()}") # strip()으로 줄바꿈 제거

2.2 내용 추가하기 ('a' 모드)

'a'(append) 모드는 기존 내용을 유지하면서 파일의 맨 끝에 새로운 내용을 추가합니다.
with open('hello.txt', 'a', encoding='utf-8') as f:
    f.write("\n새로운 내용이 추가되었습니다.")
심화

이진 파일과 파일 포인터 제어

이미지와 같은 바이너리 데이터를 복사하거나 파일의 특정 위치로 이동하는 고급 기법입니다.

3.1 이미지 파일 복사 ('rb', 'wb')

# 이진(binary) 모드에서는 encoding을 지정하지 않습니다.
with open('source.jpg', 'rb') as f_in:
    data = f_in.read()
    with open('copy.jpg', 'wb') as f_out:
        f_out.write(data)

3.2 파일 포인터 제어 (seek, tell)

- tell(): 현재 커서의 위치(바이트 단위)를 반환
- seek(offset): 해당 위치로 커서를 즉시 이동 
with open('test.txt', 'r', encoding='utf-8') as f:
    f.seek(10) # 10바이트 지점으로 점프
    print(f"현재 위치: {f.tell()}")
    print(f"그 지점의 데이터: {f.read(5)}")
