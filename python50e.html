3. 예외 처리 — 예제 231~240

예제 231. 예외가 발생하는 코드

주의
0으로 나누면 ZeroDivisionError가 발생합니다. 예외 처리를 사용하면 프로그램이 갑자기 종료되는 것을 줄일 수 있습니다.

예제 232. ZeroDivisionError 처리

try:
    result = 10 / 0
    print(result)
except ZeroDivisionError:
    print("0으로 나눌 수 없습니다.")
실행 결과
0으로 나눌 수 없습니다.
설명
try에서 오류가 발생하면 해당 예외를 처리하는 except가 실행됩니다.

예제 233. ValueError 처리

try:
    age = int("열여섯")
    print(age)
except ValueError:
    print("숫자로 변환할 수 없습니다.")
실행 결과
숫자로 변환할 수 없습니다.
설명
숫자가 아닌 문자열을 int()로 변환하면 ValueError가 발생합니다.

예제 234. 사용자 입력 예외 처리

try:
    number = int(input("숫자를 입력하세요: "))
    print(number * 2)

    print("숫자만 입력하세요.")
설명
입력값이 잘못되었을 때 프로그램이 멈추지 않도록 예외 처리를 할 수 있습니다.

예제 235. 여러 예외 처리

try:
    number = int(input("숫자 입력: "))

    print("0으로 나눌 수 없습니다.")
설명
발생할 수 있는 오류 종류에 따라 여러 개의 except를 작성할 수 있습니다.

주의
Exception은 많은 오류를 한 번에 처리하지만, 가능하면 구체적인 예외 종류를 처리하는 것이 좋습니다.

설명
else는 try 블록에서 예외가 발생하지 않았을 때 실행됩니다.

예제 243. with문으로 파일 쓰기

with open("memo.txt", "w", encoding="utf-8") as file:
    file.write("첫 번째 메모\n")
    file.write("두 번째 메모")
설명
with문을 사용하면 작업이 끝난 뒤 파일이 자동으로 닫히므로 close()를 직접 호출하지 않아도 됩니다.

예제 244. with문으로 파일 읽기

with open("memo.txt", "r", encoding="utf-8") as file:
    content = file.read()

print(content)
실행 결과 예시
첫 번째 메모
두 번째 메모
설명
파일 읽기에도 with문을 사용하는 것이 일반적으로 권장됩니다.

예제 245. 파일에 내용 추가 a 모드

with open("memo.txt", "a", encoding="utf-8") as file:
    file.write("\n세 번째 메모")
설명
"a" 모드는 기존 내용을 지우지 않고 파일 끝에 새 내용을 추가합니다.

예제 246. readline으로 한 줄 읽기

with open("memo.txt", "r", encoding="utf-8") as file:
    line = file.readline()

print(line)
설명
readline()은 파일에서 한 줄만 읽습니다.

예제 247. 파일을 한 줄씩 반복

with open("memo.txt", "r", encoding="utf-8") as file:
    for line in file:
        print(line.strip())
설명
파일 객체를 반복하면 한 줄씩 읽을 수 있습니다. strip()은 줄바꿈 문자를 제거합니다.

예제 248. writelines로 여러 줄 저장

lines = [
    "사과\n",
    "바나나\n",
    "포도\n"
]

with open("fruits.txt", "w", encoding="utf-8") as file:
file.writelines(lines)
설명
writelines()는 문자열이 들어 있는 리스트를 파일에 여러 줄로 저장할 때 사용합니다.

예제 249. CSV 파일 쓰기

import csv

rows = [
["이름", "점수"],
["민수", 80],
["지민", 95]
]

with open("scores.csv", "w", newline="", encoding="utf-8-sig") as file:
writer = csv.writer(file)
writer.writerows(rows)
설명
csv 모듈을 사용하면 표 형태의 데이터를 CSV 파일로 저장할 수 있습니다. utf-8-sig은 스프레드시트 프로그램에서 한글이 깨지는 문제를 줄이는 데 도움이 됩니다.

예제 250. CSV 파일 읽기

import csv

with open("scores.csv", "r", encoding="utf-8-sig") as file:
reader = csv.reader(file)

```
for row in reader:
    print(row)
```
실행 결과 예시
['이름', '점수']
['민수', '80']
['지민', '95']
설명
csv.reader()는 CSV 파일의 각 행을 리스트 형태로 읽어 옵니다.

5편 핵심 요약

  • 클래스는 객체를 만들기 위한 설계도이고, 객체는 클래스에서 만든 실제 데이터입니다.
  • __init__은 객체 생성 시 실행되는 생성자이며, self는 현재 객체를 의미합니다.
  • 상속을 사용하면 부모 클래스의 기능을 재사용하고 확장할 수 있습니다.
  • try-except를 사용하면 오류가 발생해도 프로그램이 안전하게 처리되도록 만들 수 있습니다.
  • with open()을 사용하면 텍스트 파일과 CSV 파일을 편리하고 안전하게 읽고 쓸 수 있습니다.
