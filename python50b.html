1. 리스트 — 예제 51~75

예제 51. 리스트 만들기

list기초 
fruits = ["사과", "바나나", "포도"]
print(fruits)
실행 결과
['사과', '바나나', '포도']
설명
리스트는 대괄호 [] 안에 여러 값을 쉼표로 구분하여 저장하는 자료형입니다.

예제 52. 리스트 인덱싱

fruits = ["사과", "바나나", "포도"]

print(fruits[0])
print(fruits[1])
print(fruits[-1])
실행 결과
사과
바나나
포도
설명
리스트의 첫 번째 요소는 인덱스 0입니다. -1은 마지막 요소를 의미합니다.

예제 53. 리스트 값 수정

fruits = ["사과", "바나나", "포도"]
fruits[1] = "딸기"

print(fruits)
실행 결과
['사과', '딸기', '포도']
설명
리스트는 변경 가능한 자료형이므로 특정 위치의 값을 수정할 수 있습니다.

예제 54. 리스트 슬라이싱

numbers = [10, 20, 30, 40, 50]

print(numbers[1:4])
print(numbers[:3])
print(numbers[3:])
실행 결과
[20, 30, 40]
[10, 20, 30]
[40, 50]
설명
[시작:끝] 범위에서 끝 인덱스는 포함하지 않습니다.

예제 55. append로 요소 추가

animals = ["고양이", "강아지"]
animals.append("토끼")

print(animals)
실행 결과
['고양이', '강아지', '토끼']
설명
append()는 리스트의 맨 뒤에 요소 하나를 추가합니다.

예제 56. insert로 원하는 위치에 추가

colors = ["빨강", "파랑"]
colors.insert(1, "노랑")

print(colors)
실행 결과
['빨강', '노랑', '파랑']
설명
insert(위치, 값)은 원하는 인덱스에 값을 삽입합니다.

예제 57. extend로 여러 요소 추가

list1 = [1, 2, 3]
list1.extend([4, 5, 6])

print(list1)
실행 결과
[1, 2, 3, 4, 5, 6]
설명
extend()는 다른 리스트의 모든 요소를 현재 리스트 뒤에 추가합니다.

예제 58. remove로 값 삭제

fruits = ["사과", "바나나", "포도"]
fruits.remove("바나나")

print(fruits)
실행 결과
['사과', '포도']
설명
remove(값)은 리스트에서 처음 발견한 해당 값을 삭제합니다.

예제 59. pop으로 마지막 값 삭제

numbers = [10, 20, 30]
removed = numbers.pop()

print(removed)
print(numbers)
실행 결과
30
[10, 20]
설명
pop()은 마지막 요소를 삭제하고, 삭제한 값을 반환합니다.

예제 60. pop으로 특정 위치 삭제

numbers = [10, 20, 30, 40]
removed = numbers.pop(1)

print(removed)
print(numbers)
실행 결과
20
[10, 30, 40]
설명
pop(인덱스)는 해당 위치의 값을 삭제하고 반환합니다.

예제 61. del로 요소 삭제

cities = ["광주", "서울", "부산"]
del cities[1]

print(cities)
실행 결과
['광주', '부산']
설명
del은 인덱스를 이용해 요소를 삭제합니다.

예제 62. 리스트 길이 확인

scores = [90, 85, 100, 78]
print(len(scores))
실행 결과
4
설명
len()은 리스트에 들어 있는 요소의 개수를 반환합니다.

예제 63. 리스트에 값이 있는지 확인

fruits = ["사과", "바나나", "포도"]

print("사과" in fruits)
print("수박" in fruits)
실행 결과
True
False
설명
in 연산자는 특정 값이 리스트에 포함되어 있는지 확인합니다.

예제 64. 리스트 합계와 최댓값

scores = [80, 95, 72, 100]

print(sum(scores))
print(max(scores))
print(min(scores))
실행 결과
347
100
72
설명
sum()은 합계, max()는 최댓값, min()은 최솟값을 구합니다.

예제 65. 리스트 평균 구하기

scores = [80, 90, 100]
average = sum(scores) / len(scores)

print(average)
실행 결과
90.0
설명
평균은 전체 합계를 요소 개수로 나누어 구합니다.

예제 66. sort로 오름차순 정렬

numbers = [5, 2, 9, 1, 7]
numbers.sort()

print(numbers)
실행 결과
[1, 2, 5, 7, 9]
설명
sort()는 원래 리스트 자체를 오름차순으로 정렬합니다.

예제 67. 내림차순 정렬

numbers = [5, 2, 9, 1, 7]
numbers.sort(reverse=True)

print(numbers)
실행 결과
[9, 7, 5, 2, 1]
설명
reverse=True를 사용하면 내림차순으로 정렬합니다.

예제 68. sorted 함수

numbers = [5, 2, 9, 1, 7]
new_numbers = sorted(numbers)

print(numbers)
print(new_numbers)
실행 결과
[5, 2, 9, 1, 7]
[1, 2, 5, 7, 9]
설명
sorted()는 원본을 바꾸지 않고 정렬된 새 리스트를 반환합니다.

예제 69. reverse로 순서 뒤집기

numbers = [1, 2, 3, 4, 5]
numbers.reverse()

print(numbers)
실행 결과
[5, 4, 3, 2, 1]
설명
reverse()는 현재 리스트의 순서를 반대로 뒤집습니다.

예제 70. 리스트 반복 출력

fruits = ["사과", "바나나", "포도"]

for fruit in fruits:
    print(fruit)
실행 결과
사과
바나나
포도
설명
for문을 사용하면 리스트의 요소를 하나씩 꺼내어 반복할 수 있습니다.

예제 71. enumerate로 번호와 함께 출력

fruits = ["사과", "바나나", "포도"]

for index, fruit in enumerate(fruits, start=1):
    print(index, fruit)
실행 결과
1 사과
2 바나나
3 포도
설명
enumerate()는 인덱스와 값을 함께 가져옵니다. start=1은 번호를 1부터 시작하게 합니다.

예제 72. 리스트 복사

original = [1, 2, 3]
copied = original.copy()

copied.append(4)

print(original)
print(copied)
실행 결과
[1, 2, 3]
[1, 2, 3, 4]
설명
copy()를 사용하면 원본과 별개의 새 리스트를 만들 수 있습니다.

예제 73. 얕은 복사 주의

original = [1, 2, 3]
copied = original

copied.append(4)

print(original)
print(copied)
실행 결과
[1, 2, 3, 4]
[1, 2, 3, 4]
주의
copied = original은 복사가 아니라 같은 리스트를 함께 가리키는 것입니다.

예제 74. 리스트 컴프리헨션

squares = [number ** 2 for number in range(1, 6)]
print(squares)
실행 결과
[1, 4, 9, 16, 25]
설명
리스트 컴프리헨션은 반복문으로 리스트를 만드는 짧고 편리한 문법입니다.

예제 75. 조건이 있는 리스트 컴프리헨션

even_numbers = [number for number in range(1, 11) if number % 2 == 0]
print(even_numbers)
실행 결과
[2, 4, 6, 8, 10]
설명
if 조건을 추가하면 조건을 만족하는 값만 리스트에 넣을 수 있습니다.

2. 튜플 — 예제 76~82

예제 76. 튜플 만들기

colors = ("빨강", "노랑", "파랑")
print(colors)
실행 결과
('빨강', '노랑', '파랑')
설명
튜플은 소괄호 ()로 여러 값을 저장합니다. 기본적으로 값을 수정할 수 없습니다.

예제 77. 튜플 인덱싱

colors = ("빨강", "노랑", "파랑")

print(colors[0])
print(colors[-1])
실행 결과
빨강
파랑
설명
튜플도 리스트처럼 인덱싱과 슬라이싱이 가능합니다.

예제 78. 튜플 슬라이싱

numbers = (10, 20, 30, 40, 50)
print(numbers[1:4])
실행 결과
(20, 30, 40)
설명
튜플 슬라이싱의 결과도 튜플입니다.

예제 79. 튜플은 수정할 수 없음

colors = ("빨강", "노랑", "파랑")

# colors[0] = "초록"

print(colors)
주의
튜플 요소를 수정하려 하면 TypeError가 발생합니다. 변경하지 않을 데이터를 저장할 때 사용합니다.

예제 80. 요소가 하나인 튜플

number = (10,)
print(number)
print(type(number))
실행 결과
(10,)
<class 'tuple'>
설명
요소가 하나인 튜플은 값 뒤에 쉼표를 반드시 붙여야 합니다.

예제 81. 튜플 언패킹

student = ("민지", 16, "광주")

name, age, city = student

print(name)
print(age)
print(city)
실행 결과
민지
16
광주
설명
튜플의 여러 값을 각각 다른 변수에 나누어 저장하는 것을 언패킹이라고 합니다.

예제 82. 변수 값 교환

a = 10
b = 20

a, b = b, a

print(a, b)
실행 결과
20 10
설명
파이썬에서는 튜플 언패킹을 이용해 임시 변수 없이 두 변수의 값을 바꿀 수 있습니다.

3. 딕셔너리 — 예제 83~93

예제 83. 딕셔너리 만들기

student = {
    "이름": "지민",
    "학년": 2,
    "점수": 95
}

print(student)
실행 결과
{'이름': '지민', '학년': 2, '점수': 95}
설명
딕셔너리는 키: 값 형태로 데이터를 저장합니다. 중괄호 {}를 사용합니다.

예제 84. 딕셔너리 값 가져오기

student = {"이름": "지민", "점수": 95}

print(student["이름"])
print(student["점수"])
실행 결과
지민
95
설명
딕셔너리는 인덱스가 아니라 키를 이용해 값을 가져옵니다.

예제 85. get으로 안전하게 값 가져오기

student = {"이름": "지민", "점수": 95}

print(student.get("이름"))
print(student.get("주소"))
print(student.get("주소", "정보 없음"))
실행 결과
지민
None
정보 없음
설명
get()은 없는 키를 조회해도 오류가 발생하지 않습니다.

예제 86. 딕셔너리 값 수정

student = {"이름": "지민", "점수": 95}
student["점수"] = 100

print(student)
실행 결과
{'이름': '지민', '점수': 100}
설명
이미 존재하는 키에 새 값을 대입하면 해당 값이 수정됩니다.

예제 87. 딕셔너리에 새 항목 추가

student = {"이름": "지민", "점수": 95}
student["학년"] = 2

print(student)
실행 결과
{'이름': '지민', '점수': 95, '학년': 2}
설명
존재하지 않는 키에 값을 대입하면 새로운 항목이 추가됩니다.

예제 88. del로 항목 삭제

student = {"이름": "지민", "점수": 95, "학년": 2}
del student["학년"]

print(student)
실행 결과
{'이름': '지민', '점수': 95}
설명
del 딕셔너리[키]로 특정 키와 값을 함께 삭제합니다.

예제 89. pop으로 항목 삭제하기

student = {"이름": "지민", "점수": 95}
score = student.pop("점수")

print(score)
print(student)
실행 결과
95
{'이름': '지민'}
설명
pop(키)는 항목을 삭제하면서 삭제한 값을 반환합니다.

예제 90. keys, values, items

student = {"이름": "지민", "점수": 95, "학년": 2}

print(student.keys())
print(student.values())
print(student.items())
설명
keys()는 키 목록, values()는 값 목록, items()는 키와 값의 쌍을 제공합니다.

예제 91. 딕셔너리 반복하기

student = {"이름": "지민", "점수": 95, "학년": 2}

for key, value in student.items():
    print(key, ":", value)
실행 결과
이름 : 지민
점수 : 95
학년 : 2
설명
items()와 반복문을 사용하면 키와 값을 함께 처리할 수 있습니다.

예제 92. 딕셔너리에 키가 있는지 확인

student = {"이름": "지민", "점수": 95}

print("이름" in student)
print("주소" in student)
실행 결과
True
False
설명
딕셔너리에서 in은 키가 존재하는지 확인합니다.

예제 93. 딕셔너리 컴프리헨션

squares = {number: number ** 2 for number in range(1, 6)}
print(squares)
실행 결과
{1: 1, 2: 4, 3: 9, 4: 16, 5: 25}
설명
딕셔너리 컴프리헨션은 반복문을 이용해 키와 값을 한 줄로 생성하는 문법입니다.

4. 집합 — 예제 94~100

예제 94. 집합 만들기

numbers = {1, 2, 3, 4, 5}
print(numbers)
설명
집합은 중괄호를 사용하지만, 딕셔너리처럼 키와 값의 쌍이 아닌 값만 저장합니다.

예제 95. 집합의 중복 제거

numbers = {1, 2, 2, 3, 3, 3, 4}
print(numbers)
실행 결과 예시
{1, 2, 3, 4}
설명
집합은 같은 값을 한 번만 저장합니다. 출력 순서는 일정하지 않을 수 있습니다.

예제 96. add로 요소 추가

animals = {"고양이", "강아지"}
animals.add("토끼")

print(animals)
설명
add()는 집합에 요소 하나를 추가합니다.

예제 97. update로 여러 요소 추가

animals = {"고양이", "강아지"}
animals.update(["토끼", "햄스터"])

print(animals)
설명
update()는 리스트, 튜플, 집합 등에 있는 여러 요소를 한 번에 추가합니다.

예제 98. remove와 discard

animals = {"고양이", "강아지", "토끼"}

animals.remove("토끼")
animals.discard("햄스터")

print(animals)
실행 결과 예시
{'고양이', '강아지'}
주의
remove()는 없는 값을 삭제하면 오류가 나지만, discard()는 오류 없이 지나갑니다.

예제 99. 합집합

class_a = {"민수", "지민", "서연"}
class_b = {"지민", "도윤", "하늘"}

print(class_a | class_b)
print(class_a.union(class_b))
설명
합집합은 두 집합의 모든 요소를 중복 없이 합친 결과입니다.

예제 100. 교집합과 차집합

class_a = {"민수", "지민", "서연"}
class_b = {"지민", "도윤", "하늘"}

print(class_a & class_b)
print(class_a - class_b)
실행 결과 예시
{'지민'}
{'민수', '서연'}
설명
&는 교집합, -는 차집합을 구합니다.

2편 핵심 요약

  • 리스트는 순서가 있고 수정할 수 있는 자료형입니다.
  • 튜플은 순서가 있지만 수정할 수 없는 자료형입니다.
  • 딕셔너리는 키와 값의 쌍으로 데이터를 저장합니다.
  • 집합은 중복을 허용하지 않으며 합집합·교집합·차집합 연산에 유용합니다.
  • 자료구조를 적절히 선택하면 데이터를 더 효율적으로 저장하고 처리할 수 있습니다.
