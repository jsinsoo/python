10차시: 딕셔너리(Dictionary) 마스터

데이터에 '이름'을 붙여 관리하는 Key-Value 구조를 완벽히 이해합니다.

기초

딕셔너리 생성 및 접근

딕셔너리는 중괄호 {}를 사용하며, 각 요소는 키: 값의 형태를 가집니다.

Key
(이름, ID 등)
➔
Value
(실제 데이터)
# 학생 정보 딕셔너리
student = {
    "name": "홍길동",
    "id": 202301,
    "major": "컴퓨터공학"
}

# 값 가져오기
print(student["name"]) 
# 안전하게 가져오기 (키가 없어도 에러 발생 안 함)
print(student.get("age", "정보 없음"))
응용

데이터 조작 및 순회

딕셔너리의 데이터를 추가, 수정, 삭제하고 반복문을 통해 모든 데이터를 확인합니다.

person = {"name": "김철수"}

# 추가 및 수정
person["age"] = 30
person["age"] = 31 # 기존 값 덮어쓰기

# 모든 키-값 확인하기
for key, value in person.items():
    print(f"{key}: {value}")

# 특정 키 존재 여부 확인
if "name" in person:
    print("이름 정보가 있습니다.")
심화

복합 구조 및 정렬

실제 API 데이터와 유사한 중첩 구조와 데이터를 정렬하는 방법을 다룹니다.

💡 실무 팁: 딕셔너리와 리스트를 조합하면 JSON과 같은 복잡한 계층형 데이터를 자유자재로 다룰 수 있습니다. 
# 중첩된 데이터 구조
api_response = {
    "data": [
        {"id": 101, "name": "상품A", "tags": ["전자", "IT"]},
        {"id": 102, "name": "상품B", "tags": ["가구"]}
    ]
}

# 값 추출
first_tag = api_response["data"][0]["tags"][1] # "IT"

# 값(Value) 기준으로 정렬하기
scores = {"수학": 100, "영어": 80, "국어": 95}
sorted_res = sorted(scores.items(), key=lambda x: x[1], reverse=True)
print(sorted_res) # [('수학', 100), ('국어', 95), ('영어', 80)]
