12차시: 문자열 처리(String) 마스터

텍스트 데이터를 자유자재로 다루는 파이썬의 강력한 문자열 기술을 배웁니다.

기초: 생성과 슬라이싱

문자열의 길이를 구하고, 원하는 부분만 추출하는 기본 기술입니다.

1.1 인덱싱과 슬라이싱

슬라이싱 [시작:끝]에서 끝 인덱스는 포함되지 않음을 항상 기억하세요!
text = "Python Programming"

# 슬라이싱: 0번부터 5번 인덱스까지 (6번 미포함)
print(text[0:6])    # "Python"
# 뒤에서부터 추출
print(text[-11:])   # "Programming"
# 거꾸로 뒤집기
print(text[::-1])   # "gnimmargorP nohtyP"
응용: 핵심 문자열 메서드

자주 사용되는 내장 함수들을 통해 데이터를 정제하는 방법을 익힙니다.

2.1 결합과 분리

.split()은 리스트로 쪼개고, .join()은 문자열로 합칩니다.
# 문자열 쪼개기
data = "apple,banana,cherry"
fruits = data.split(",") # ['apple', 'banana', 'cherry']

# 문자열 합치기
new_data = "-".join(fruits)
print(new_data) # "apple-banana-cherry"

2.2 검색 및 검사

filename = "report.pdf"

# 특정 문자로 끝나는지 확인 (확장자 체크)
if filename.endswith(".pdf"):
    print("PDF 파일이 맞습니다.")

# 문자 위치 찾기 (없으면 -1 반환)
print(filename.find("port")) # 2
심화: 포매팅과 정규표현식

가독성 높은 출력 방법인 f-string과 복잡한 패턴 매칭을 다룹니다.

3.1 f-string (Python 3.6+)

변수뿐만 아니라 함수 호출이나 연산도 문자열 안에서 직접 수행할 수 있습니다.
name = "이재명"
score = 95
print(f"[{name}]님의 점수는 {score}점이며, 등급은 {'A' if score >= 90 else 'B'}입니다.")

3.2 정규표현식 (re 모듈)

import re

text = "문의사항은 010-1234-5678로 연락주세요."
# 전화번호 패턴 매칭
phone_pattern = r"\d{3}-\d{3,4}-\d{4}"
phone = re.search(phone_pattern, text)

if phone:
    print(f"찾은 번호: {phone.group()}")
