13차시: 파일 입출력(File I/O) 마스터하기
파일 입출력은 프로그램의 데이터를 영구적으로 저장하거나(출력), 파일에 저장된 데이터를 프로그램으로 불러오는(입력) 필수적인 기능입니다. Python에서는 간단한 내장 함수를 통해 파일을 쉽게 다룰 수 있습니다. 기초부터 심화까지 다양한 예제를 통해 파일 처리 방법을 알아보겠습니다.

1. 기초 파이썬 예시 코드 (5개)
파일을 열고, 쓰고, 읽는 가장 기본적인 방법을 익힙니다.

1) 새 파일 생성 및 내용 쓰기 ('w' 모드)
'w' (write) 모드는 파일이 없으면 새로 생성하고, 파일이 이미 존재하면 기존 내용을 모두 덮어씁니다.

# 'w' 모드로 'hello.txt' 파일을 열고 파일 객체를 f에 할당
# with 구문은 파일 사용이 끝나면 자동으로 닫아줍니다.
with open('hello.txt', 'w', encoding='utf-8') as f:
    f.write("안녕하세요, 파일 입출력의 세계에 오신 것을 환영합니다.\n")
    f.write("이것은 두 번째 줄입니다.")

# with 블록이 끝나면 파일은 자동으로 저장되고 닫힙니다.
print("hello.txt 파일이 생성되고 내용이 작성되었습니다.")
2) 파일 내용 전체 읽기 ('r' 모드)
'r' (read) 모드는 파일의 내용을 읽기 위해 사용됩니다. 파일이 존재하지 않으면 FileNotFoundError가 발생합니다.

try:
    with open('hello.txt', 'r', encoding='utf-8') as f:
        content = f.read() # 파일 전체 내용을 하나의 문자열로 읽어옴
        print("--- 파일 내용 ---")
        print(content)
except FileNotFoundError:
    print("hello.txt 파일을 찾을 수 없습니다.")
3) 파일 끝에 내용 추가하기 ('a' 모드)
'a' (append) 모드는 파일이 존재하면 기존 내용의 맨 끝에 새로운 내용을 추가합니다. 파일이 없으면 새로 생성합니다.

with open('hello.txt', 'a', encoding='utf-8') as f:
    f.write("\n이 내용은 파일 끝에 추가되었습니다.")

print("hello.txt 파일에 내용이 추가되었습니다.")

# 추가된 내용 확인
with open('hello.txt', 'r', encoding='utf-8') as f:
    print("\n--- 추가 후 파일 내용 ---")
    print(f.read())
4) 파일 한 줄씩 읽기 (readline)
readline() 메서드는 파일의 내용을 한 번에 한 줄씩 읽어옵니다. 파일의 끝에 도달하면 빈 문자열을 반환합니다.

print("\n--- 파일 한 줄씩 읽기 ---")
with open('hello.txt', 'r', encoding='utf-8') as f:
    while True:
        line = f.readline()
        if not line: # 더 이상 읽을 줄이 없으면 반복 중단
            break
        print(line.strip()) # strip()으로 양쪽 끝의 공백 및 줄바꿈 문자 제거
5) 파일 모든 줄을 리스트로 읽기 (readlines)
readlines() 메서드는 파일의 모든 줄을 읽어 각 줄을 요소로 하는 리스트를 반환합니다.

print("\n--- 파일 모든 줄을 리스트로 읽기 ---")
with open('hello.txt', 'r', encoding='utf-8') as f:
    lines = f.readlines()
    print(lines)
    # 각 줄을 반복하며 처리
    for i, line in enumerate(lines):
        print(f"{i+1}번째 줄: {line.strip()}")
2. 응용 파이썬 예시 코드 (5개)
파일 입출력을 실제 데이터 처리 시나리오에 적용하는 방법을 알아봅니다.

1) 리스트의 내용을 파일에 쓰기
subjects = ['국어', '영어', '수학', '과학']

with open('subjects.txt', 'w', encoding='utf-8') as f:
    for subject in subjects:
        f.write(subject + '\n') # 각 항목을 쓰고 줄바꿈 문자를 추가

print("subjects.txt 파일이 생성되었습니다.")
2) CSV 파일 형식의 데이터 읽고 처리하기
간단한 이름,점수 형식의 CSV 파일을 읽어 총점과 평균을 계산합니다.

# 예시용 CSV 파일 생성
with open('scores.csv', 'w', encoding='utf-8') as f:
    f.write("홍길동,90\n")
    f.write("이순신,95\n")
    f.write("세종대왕,100\n")

total_score = 0
count = 0
with open('scores.csv', 'r', encoding='utf-8') as f:
    for line in f:
        name, score = line.strip().split(',')
        total_score += int(score)
        count += 1
        print(f"이름: {name}, 점수: {score}")

print(f"\n총점: {total_score}, 평균: {total_score / count:.2f}")
3) 파일 내용 복사하기
한 파일의 내용을 읽어 다른 파일에 그대로 복사합니다.

source_file = 'hello.txt'
dest_file = 'hello_copy.txt'

try:
    with open(source_file, 'r', encoding='utf-8') as sf:
        content = sf.read()
    
    with open(dest_file, 'w', encoding='utf-8') as df:
        df.write(content)
    
    print(f"'{source_file}'의 내용이 '{dest_file}'에 복사되었습니다.")
except FileNotFoundError:
    print(f"원본 파일 '{source_file}'을 찾을 수 없습니다.")

4) 파일에서 특정 단어 개수 세기
word_to_count = "파일"
count = 0

try:
    with open('hello.txt', 'r', encoding='utf-8') as f:
        content = f.read()
        words = content.split() # 공백 기준으로 단어 분리
        for word in words:
            if word_to_count in word:
                count += 1
    
    print(f"'hello.txt'에서 '{word_to_count}' 단어의 등장 횟수: {count}")
except FileNotFoundError:
    print("검색할 파일을 찾을 수 없습니다.")
5) 로그 파일 시뮬레이션
프로그램 실행 시점의 로그를 시간과 함께 파일에 기록합니다.

import datetime

def log_message(message):
    now = datetime.datetime.now()
    timestamp = now.strftime("%Y-%m-%d %H:%M:%S")
    log_entry = f"[{timestamp}] {message}\n"
    
    with open('app.log', 'a', encoding='utf-8') as f:
        f.write(log_entry)

# 로그 기록
log_message("프로그램 시작")
log_message("데이터 처리 중...")
log_message("프로그램 종료")

print("app.log 파일에 로그가 기록되었습니다.")
3. 심화 파이썬 예시 코드 (5개)
파일 시스템, 인코딩, 예외 처리 등 고급 주제를 다룹니다.

1) try-except-finally로 파일 예외 처리
with 구문이 권장되지만, finally를 통해 파일 닫기를 보장하는 전통적인 방식도 이해해두면 좋습니다.

f = None # f 변수를 바깥에 선언
try:
    f = open('non_existent_file.txt', 'r', encoding='utf-8')
    print(f.read())
except FileNotFoundError:
    print("오류: 파일을 찾을 수 없습니다.")
except Exception as e:
    print(f"알 수 없는 오류 발생: {e}")
finally:
    if f: # 파일이 성공적으로 열렸을 경우에만 닫기
        f.close()
        print("파일 객체가 닫혔습니다.")
2) os 모듈로 파일 존재 여부 확인 후 작업하기
파일을 열기 전에 os.path.exists()로 존재 여부를 먼저 확인하여 오류를 예방할 수 있습니다.

import os

file_path = 'app.log'

if os.path.exists(file_path):
    with open(file_path, 'r', encoding='utf-8') as f:
        print(f"'{file_path}' 파일의 첫 50자: {f.read(50)}...")
else:
    print(f"'{file_path}' 파일이 존재하지 않습니다.")
3) 바이너리 파일 다루기 (이미지 복사 예제)
텍스트가 아닌 이미지, 동영상 등의 바이너리 파일은 'rb'(read binary), 'wb'(write binary) 모드를 사용합니다.

# 이 코드를 실행하려면 'source_image.jpg' 파일이 필요합니다.
# 직접 가지고 있는 jpg 파일을 코드 파일과 같은 위치에 두고 이름을 변경하세요.
source_image = 'source_image.jpg'
dest_image = 'copied_image.jpg'

try:
    with open(source_image, 'rb') as f_in:
        with open(dest_image, 'wb') as f_out:
            f_out.write(f_in.read())
    print(f"이미지 파일 '{source_image}'가 '{dest_image}'로 복사되었습니다.")
except FileNotFoundError:
    print(f"'{source_image}' 파일을 찾을 수 없습니다. 같은 폴더에 이미지 파일을 준비해주세요.")
4) 특정 위치에서 읽고 쓰기 (seek, tell)
seek()은 파일 내에서 원하는 위치(offset)로 커서를 이동시키고, tell()은 현재 커서의 위치를 알려줍니다.

with open('seek_test.txt', 'w', encoding='utf-8') as f:
    f.write("0123456789abcdef")

with open('seek_test.txt', 'r+', encoding='utf-8') as f:
    f.seek(10) # 10번 인덱스 위치로 커서 이동
    data = f.read(4) # 4글자 읽기
    print(f"10번 위치에서 읽은 데이터: {data}") # 출력: abcd

    print(f"현재 커서 위치: {f.tell()}") # 출력: 14

    f.seek(0) # 파일의 맨 처음으로 이동
    f.write("####") # 앞 4글자를 '#'으로 덮어씀

# 결과 확인
with open('seek_test.txt', 'r', encoding='utf-8') as f:
    print(f"수정된 파일 내용: {f.read()}") # 출력: ####456789abcdef
5) 윈도우 환경의 cp949 인코딩 파일 처리
웹이나 최신 시스템은 대부분 utf-8을 사용하지만, 오래된 윈도우 시스템에서 생성된 한글 파일은 cp949 인코딩인 경우가 많습니다.

text_cp949 = "이것은 윈도우 기본 인코딩(cp949)으로 작성된 텍스트입니다."

# cp949로 파일 쓰기
with open('legacy.txt', 'w', encoding='cp949') as f:
    f.write(text_cp949)

# cp949로 파일 읽기
try:
    with open('legacy.txt', 'r', encoding='cp949') as f:
        content = f.read()
        print("cp949 인코딩 파일 읽기 성공:")
        print(content)
except UnicodeDecodeError:
    print("인코딩이 맞지 않아 파일을 읽을 수 없습니다.")
