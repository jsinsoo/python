11차시: 함수(Functions) 마스터

반복되는 코드를 묶어 관리하는 효율적 프로그래밍의 첫걸음입니다.

기초: 함수의 정의와 호출

함수는 입력(매개변수)을 받아 작업을 수행하고 출력(반환값)을 내보냅니다.

1.1 기본 구조

함수를 만들 때는 def 키워드를 사용하며, 작업이 끝나면 return으로 결과를 돌려줍니다.
# 사각형 넓이와 둘레 계산
def calculate_rectangle(width, height):
    area = width * height
    perimeter = 2 * (width + height)
    return area, perimeter # 여러 값 반환 시 튜플로 묶임

a, p = calculate_rectangle(10, 5)
print(f"넓이: {a}, 둘레: {p}")
응용: 매개변수의 활용

가변 인자와 키워드 인자를 사용하여 유연한 함수를 설계할 수 있습니다.

2.1 가변 인자 (*args, **kwargs)

# 개수가 정해지지 않은 숫자들의 합
def sum_all(*args):
    return sum(args)

print(sum_all(1, 2, 3, 4, 5)) # 15

# 키워드 가변 인자
def print_info(**kwargs):
    for key, value in kwargs.items():
        print(f"{key}: {value}")

print_info(name="이재명", job="개발자")
심화: 고급 함수 기법

함수를 인자로 전달하거나, 기능을 확장하는 데코레이터를 학습합니다.

3.1 클로저 (Closure)

내부 함수가 외부 함수의 변수를 기억하고 있는 상태를 말합니다.
def outer_func(base):
    def inner_func(x):
        return base * x
    return inner_func

times_5 = outer_func(5)
print(times_5(3)) # 15 (base인 5를 기억함)

3.2 데코레이터 (Decorator)

import time

def timer(func):
    def wrapper(*args, **kwargs):
        start = time.time()
        result = func(*args, **kwargs)
        print(f"실행 시간: {time.time() - start:.5f}초")
        return result
    return wrapper

@timer
def long_task():
    time.sleep(1)

long_task()
