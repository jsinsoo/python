1. 데코레이터 Decorator

기존 함수를 수정하지 않고 기능을 덧붙입니다.

예시 코드

def log(func):
    def wrapper():
        print("실행 시작")
        result = func()
        print("실행 종료")
        return result
    return wrapper

@log
def hello():
    return "Hello Python"

print(hello())

출력 결과

실행 시작
실행 종료
Hello Python

2. 클로저 Closure

내부 함수가 바깥 함수의 변수를 기억합니다.

예시 코드

def outer(x):
    def inner(y):
        return x + y
    return inner

add10 = outer(10)

print(add10(5))
print(add10(20))

출력 결과

15
30

3. 제너레이터 Generator

yield로 값을 필요할 때 하나씩 생성합니다.

예시 코드

def count_up_to(n):
    i = 1
    while i <= n:
        yield i
        i += 1

for number in count_up_to(5):
    print(number)

출력 결과

1
2
3
4
5

예시 코드

5. *args와 **kwargs

개수가 정해지지 않은 인수를 받을 때 사용합니다.

예시 코드

def student_info(*args, **kwargs):
    print("과목:", args)
    print("정보:", kwargs)

student_info("국어", "수학", "정보", name="민수", grade=1)

출력 결과

과목: ('국어', '수학', '정보')
정보: {'name': '민수', 'grade': 1}

6. 컨텍스트 매니저 Context Manager

with 문으로 시작과 종료 작업을 자동 처리합니다.

예시 코드

class StudyRoom:
    def __enter__(self):
        print("공부방 입장")
        return self

    def __exit__(self, exc_type, exc_value, traceback):
        print("공부방 퇴장")

with StudyRoom():
    print("파이썬 공부 중")

출력 결과

공부방 입장
파이썬 공부 중
공부방 퇴장

7. 이터레이터 Iterator

for 문이 값을 하나씩 꺼내는 원리를 직접 구현합니다.

예시 코드

class Counter:
    def __init__(self, end):
        self.current = 0
        self.end = end

    def __iter__(self):
        return self

    def __next__(self):
        self.current += 1
        if self.current > self.end:
            raise StopIteration
        return self.current

for number in Counter(5):
    print(number)

출력 결과

1
2
3
4
5

8. 함수형 프로그래밍 Lambda, Map, Filter

함수를 값처럼 다루어 데이터를 변환하거나 걸러냅니다.

예시 코드

numbers = [1, 2, 3, 4, 5, 6]

result = list(
    map(lambda x: x * x,
        filter(lambda x: x % 2 == 0, numbers))
)

print(result)

출력 결과

[4, 16, 36]

9. 메타클래스 Metaclass

클래스를 만드는 클래스를 정의합니다.

예시 코드

class AddSchoolName(type):
    def __new__(cls, name, bases, attrs):
        attrs["school"] = "전북과학고"
        return super().__new__(cls, name, bases, attrs)

class Student(metaclass=AddSchoolName):
    pass

print(Student.school)

출력 결과

전북과학고

10. 비동기 프로그래밍 Async/Await

기다리는 시간이 많은 작업을 효율적으로 처리합니다.

예시 코드

import asyncio

async def study():
    print("공부 시작")
    await asyncio.sleep(1)
    print("공부 완료")

asyncio.run(study())

공부 완료
