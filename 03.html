3차시: 파이썬 입출력 함수 학습 패키지

데이터의 '입력'과 '출력' 및 파일 처리의 핵심 예제를 학습합니다.

1. 기초 파이썬 예제

사용자와 상호작용하고 기본적인 파일 읽기/쓰기를 다룹니다.

1. 화면 출력과 사용자 입력

print()로 출력하고 input()으로 데이터를 입력받습니다. 입력값은 항상 문자열임을 주의하세요.

# 화면 출력
print("Hello, Python!")

# 사용자 입력 (항상 문자열로 반환됨)
name = input("이름을 입력하세요: ")
print(f"안녕하세요, {name}님!")

# 숫자형으로 변환 후 계산
year_int = int(input("태어난 연도를 입력하세요: "))
age = 2026 - year_int + 1
print(f"당신의 한국 나이는 {age}세입니다.")

2. 기본적인 파일 처리 (w/r 모드)

'w' 모드는 덮어쓰기, 'r' 모드는 읽기 전용입니다. 작업 후에는 반드시 .close()를 호출해야 합니다.

# 파일 쓰기 (w)
f = open("memo.txt", "w", encoding="utf-8")
f.write("파이썬으로 파일 쓰기 연습\n첫 번째 줄입니다.")
f.close() 

# 파일 읽기 (r)
f = open("memo.txt", "r", encoding="utf-8")
content = f.read()
f.close()
print(content)

2. 응용 파이썬 예제

더 안전한 파일 처리와 효율적인 출력 옵션을 다룹니다.

1. with 구문을 이용한 안전한 파일 처리

with 블록을 사용하면 close()를 직접 호출하지 않아도 자동으로 파일이 닫힙니다.

with open("memo_with.txt", "w", encoding="utf-8") as f:
    f.write("with 구문은 자동으로 파일을 닫아주어 안전합니다.")

# 대용량 파일은 한 줄씩 읽는 것이 효율적입니다.
with open("memo_with.txt", "r", encoding="utf-8") as f:
    for line in f:
        print(line.strip())

2. print() 함수의 세부 옵션

sep(구분자)와 end(끝 문자) 옵션을 통해 출력 형식을 제어할 수 있습니다.

# sep: 여러 항목 사이의 문자 지정 (기본: 공백)
print("사과", "바나나", "딸기", sep=", ")

# end: 출력 마지막 문자 지정 (기본: 줄바꿈)
print("첫 번째 줄", end=" --- ")
print("두 번째 줄")

3. 심화 파이썬 예제

CSV, JSON 형식의 데이터 처리와 표준 오류 스트림을 다룹니다.

1. CSV 및 JSON 데이터 처리

엑셀과 호환되는 CSV 형식과 웹 표준인 JSON 형식을 저장하고 불러옵니다.

import csv
import json

# CSV 파일 쓰기
data = [["이름", "부서"], ["홍길동", "개발팀"]]
with open("company.csv", "w", newline="", encoding="utf-8-sig") as f:
    writer = csv.writer(f)
    writer.writerows(data)

# JSON 데이터 저장 (딕셔너리 구조)
my_data = {"name": "홍길동", "skills": ["Python", "AI"]}
with open("profile.json", "w", encoding="utf-8") as f:
    json.dump(my_data, f, indent=4, ensure_ascii=False)

2. 표준 오류(stderr) 출력

중요한 오류 메시지는 일반 출력과 분리하여 표준 오류 스트림으로 보냅니다.

import sys

print("일반 정보 메시지") # 표준 출력 (stdout)
print("치명적인 오류 발생!", file=sys.stderr) # 표준 오류 (stderr)
