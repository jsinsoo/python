Basics

데이터 구조의 이해

Pandas는 1차원 배열 형태인 Series와 2차원 표 형태인 DataFrame을 사용합니다.

Index Column 1 (Series) Column 2
0Data AValue 1
1Data BValue 2
 
import pandas as pd

# DataFrame 생성 (딕셔너리 활용)
data = {
    '이름': ['홍길동', '윤봉길', '안중근'],
    '점수': [95, 88, 92]
}
df = pd.DataFrame(data)
print(df.head(2)) # 상위 2개 행 확인
Intermediate

데이터 필터링 및 그룹화

조건에 맞는 데이터를 추출하거나 특정 기준(Group)으로 통계량을 계산합니다.

# 1. 조건 필터링: 점수가 90점 이상인 사람만
high_scorers = df[df['점수'] >= 90]

# 2. 그룹화: 도시별 평균 온도 계산
df_city = pd.DataFrame({
    '도시': ['서울', '부산', '서울', '부산'],
    '온도': [25, 28, 26, 29]
})
avg_temp = df_city.groupby('도시')['온도'].mean()
print(avg_temp)
💡 Pivot Table 활용: 엑셀의 피벗 테이블처럼 데이터를 재구조화하여 다차원 분석을 수행할 수 있습니다.
Advanced

함수 적용과 데이터 가공

.apply()와 lambda를 사용하여 복잡한 계산 로직을 모든 행에 일괄 적용합니다.

# 새로운 열 추가: 점수에 따른 등급 부여
def get_grade(score):
    if score >= 90: return 'A'
    else: return 'B'

df['등급'] = df['점수'].apply(get_grade)

# 람다(lambda) 함수로 간결하게 처리
df['평균대비'] = df['점수'].apply(lambda x: x - df['점수'].mean())
print(df)
