조건문 응용 파이썬 예시 코드

실생활에서 접할 수 있는 다양한 상황을 조건문(if, elif, else)으로 해결하는 응용 예제 10가지입니다. 기초적인 연령 제한부터 복잡한 예외 처리까지 단계별로 구성되어 있습니다.

[기초 응용 예시 코드 5개]

1. 놀이공원 입장료 계산기

age = int(input("나이를 입력하세요: "))
price = 0

if age < 8 or age >= 65:
    price = 0
elif age < 14:
    price = 5000
elif age < 20:
    price = 8000
else:
    price = 10000

print(f"입장료는 {price}원입니다.")
핵심: or 논리 연산자와 여러 elif를 사용하여 다양한 연령대에 따른 요금 부과 조건을 구현합니다.

2. 간단한 로그인 프로그램

# 미리 저장된 아이디와 비밀번호
saved_id = "python"
saved_pw = "1234"

user_id = input("아이디를 입력하세요: ")
user_pw = input("비밀번호를 입력하세요: ")

if user_id == saved_id and user_pw == saved_pw:
    print("로그인 성공!")
elif user_id == saved_id:
    print("비밀번호가 틀렸습니다.")
else:
    print("존재하지 않는 아이디입니다.")
핵심: and 논리 연산자를 사용하여 아이디와 비밀번호가 모두 일치하는 경우를 확인하고, elif를 통해 아이디만 일치하는 경우를 구분합니다.

3. 홀수와 짝수 판별기

number = int(input("정수를 입력하세요: "))

if number % 2 == 0:
    print(f"{number}은(는) 짝수입니다.")
else:
    print(f"{number}은(는) 홀수입니다.")
핵심: 나머지 연산자(%)를 사용하여 숫자를 2로 나눈 나머지가 0인지 확인하여 홀짝을 판별합니다.

4. 학점 변환 프로그램

score = int(input("점수를 입력하세요 (0~100): "))

if 90 <= score <= 100:
    grade = "A"
elif 80 <= score < 90:
    grade = "B"
elif 70 <= score < 80:
    grade = "C"
else:
    grade = "F"

print(f"당신의 학점은 {grade}입니다.")
핵심: 파이썬 특유의 연속 비교(90 <= score <= 100)를 활용하여 점수 범위를 직관적으로 설정합니다.

5. BMI(체질량지수) 계산 및 비만도 판정

height = float(input("키(cm)를 입력하세요: ")) / 100
weight = float(input("몸무게(kg)를 입력하세요: "))

bmi = weight / (height * height)
print(f"당신의 BMI 지수는 {bmi:.2f}입니다.")

if bmi < 18.5:
    print("저체중입니다.")
elif bmi < 23:
    print("정상입니다.")
elif bmi < 25:
    print("과체중입니다.")
else:
    print("비만입니다.")
핵심: 수치 계산 결과(bmi)를 바탕으로 다중 조건문을 구성하여 상태를 분류합니다.

[심화 응용 예시 코드 5개]

6. 가위바위보 게임

import random

choices = ["가위", "바위", "보"]
computer = random.choice(choices)
user = input("가위, 바위, 보 중 하나를 선택하세요: ")

print(f"컴퓨터: {computer}, 사용자: {user}")

if user == computer:
    print("비겼습니다!")
elif (user == "가위" and computer == "보") or \
     (user == "바위" and computer == "가위") or \
     (user == "보" and computer == "바위"):
    print("이겼습니다!")
else:
    print("졌습니다.")
핵심: random 모듈과 복합 논리 연산자(and, or)를 사용하여 게임의 승패 시나리오를 정의합니다.

7. 윤년 판별기

year = int(input("연도를 입력하세요: "))

if (year % 4 == 0 and year % 100 != 0) or (year % 400 == 0):
    print(f"{year}년은 윤년입니다.")
else:
    print(f"{year}년은 평년입니다.")
핵심: 복잡한 논리 조건을 괄호와 연산자 우선순위를 활용하여 한 줄의 조건식으로 표현합니다.

8. 사칙연산이 가능한 계산기

num1 = float(input("첫 번째 숫자를 입력하세요: "))
op = input("연산자(+, -, *, /)를 입력하세요: ")
num2 = float(input("두 번째 숫자를 입력하세요: "))

result = 0
error = False

if op == '+':
    result = num1 + num2
elif op == '-':
    result = num1 - num2
elif op == '*':
    result = num1 * num2
elif op == '/':
    if num2 == 0:
        print("오류: 0으로 나눌 수 없습니다.")
        error = True
    else:
        result = num1 / num2
else:
    print("오류: 잘못된 연산자입니다.")
    error = True

if not error:
    print(f"결과: {num1} {op} {num2} = {result}")
핵심: 문자열(연산자)을 조건으로 사용하고, 중첩 조건문을 통해 '0으로 나누는 경우' 같은 예외 상황을 처리합니다.

9. 리스트 내 특정 요소 존재 여부 확인

fruits = ["사과", "바나나", "딸기", "포도"]
my_fruit = input("좋아하는 과일은 무엇인가요? ")

if my_fruit in fruits:
    print(f"와! 저도 {my_fruit}를(을) 좋아해요.")
else:
    print(f"{my_fruit}도 맛있겠네요. 하지만 제가 가진 목록에는 없어요.")
핵심: 멤버십 연산자(in)를 활용하여 데이터 집합 안에 특정 값이 있는지 간결하게 확인합니다.

10. 사용자 이름에 따른 맞춤형 인사

# 홍길동, 구운몽, 인수샘 등에 따른 역할 구분
name = input("이름을 입력하세요: ")

if name == "인수샘":
    print("안녕하세요, 선생님! 오늘 수업도 화이팅입니다.")
elif name in ["홍길동", "구운몽"]:
    print(f"반가워요, {name} 학생! 공부 열심히 하세요.")
else:
    print(f"환영합니다, {name}님!")
핵심: 특정 문자열 일치 여부와 리스트 포함 여부를 조건으로 사용하여 사용자별 맞춤 피드백을 구현합니다.
