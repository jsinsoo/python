6차시: 조건문 응용 마스터

실생활 문제를 해결하는 10가지 파이썬 조건문 시나리오

기초 응용 예제 (5개)

1. 놀이공원 입장료 계산기

age = int(input("나이를 입력하세요: "))
price = 0

if age < 8 or age >= 65:
    price = 0
elif age < 14:
    price = 5000
elif age < 20:
    price = 8000
else:
    price = 10000

print(f"입장료는 {price}원입니다.")
핵심: or 논리 연산자와 여러 개의 elif를 조합하여 복잡한 연령대별 요금 체계를 간단히 구현합니다.

2. 로그인 프로그램

saved_id = "python"
saved_pw = "1234"

user_id = input("아이디: ")
user_pw = input("비밀번호: ")

if user_id == saved_id and user_pw == saved_pw:
    print("로그인 성공!")
elif user_id == saved_id:
    print("비밀번호가 틀렸습니다.")
else:
    print("존재하지 않는 아이디입니다.")
핵심: and 연산자로 두 조건을 동시에 확인하고, elif를 통해 단계별 오류 메시지를 제공합니다.
심화 응용 예제 (5개)

8. 간단한 사칙연산 계산기

num1 = float(input("첫 번째 숫자: "))
op = input("연산자(+, -, *, /): ")
num2 = float(input("두 번째 숫자: "))

if op == '+':
    result = num1 + num2
elif op == '/':
    if num2 == 0:
        print("오류: 0으로 나눌 수 없습니다.")
    else:
        result = num1 / num2
# ... (생략)
핵심: 중첩 조건문을 활용하여 나눗셈 시 분모가 0이 되는 예외 상황을 안전하게 처리합니다.

9. 리스트 내 요소 존재 여부

fruits = ["사과", "바나나", "딸기"]
my_fruit = input("좋아하는 과일은? ")

if my_fruit in fruits:
    print(f"와! 저도 {my_fruit}를 좋아해요.")
else:
    print("아쉽네요, 목록에 없는 과일입니다.")
핵심: in 멤버십 연산자를 활용하여 데이터 집합에 특정 값이 포함되어 있는지 간결하게 판단합니다.
