8차시: while 반복문 마스터

조건이 참(True)인 동안 멈추지 않는 강력한 제어 흐름을 배웁니다.

1. 기초: while문의 구조

for 문이 '횟수' 중심이라면, while 문은 '상태(조건)' 중심입니다.

[Image of while loop flowchart in programming]

1.1 기본적인 카운터

반복할 때마다 조건을 변화시켜 무한 루프에 빠지지 않게 주의해야 합니다.
count = 1
while count <= 5:
    print(f"{count}번째 실행 중...")
    count += 1 # 이 코드가 없으면 무한히 실행됩니다!

1.2 특정 입력까지 반복

prompt = "종료하려면 'exit'를 입력하세요: "
message = ""

while message.lower() != 'exit':
    message = input(prompt)
    print(f"입력값: {message}")
2. 응용: 흐름 제어(break, continue)

반복문 내부에서 실행 흐름을 강제로 바꾸는 기법들을 학습합니다.

2.1 break를 이용한 무한 루프 탈출

while True는 의도적인 무한 루프를 만들 때 사용하며, break로 탈출구를 만듭니다.
while True:
    num = int(input("숫자를 입력하세요 (0을 입력하면 종료): "))
    if num == 0:
        print("프로그램을 종료합니다.")
        break
    print(f"입력한 숫자의 제곱: {num**2}")
3. 심화: 견고한 프로그래밍

알고리즘 구현 및 예외 처리를 결합한 실전적인 while문 사용법입니다.

3.1 유클리드 호제법 (최대공약수)

a, b = 48, 18
temp_a, temp_b = a, b

while temp_b != 0:
    temp_a, temp_b = temp_b, temp_a % temp_b

print(f"{a}와 {b}의 최대공약수: {temp_a}")

3.2 예외 처리와 결합 (try-except)

사용자가 올바른 값을 입력할 때까지 무한히 재요청하는 패턴입니다.
while True:
    try:
        age = int(input("나이를 숫자로만 입력하세요: "))
        break # 정수 변환 성공 시 루프 탈출
    except ValueError:
        print("잘못된 입력입니다. 다시 시도하세요.")
