8차시: while 반복문 학습 패키지

안녕하세요! 이번 시간에는 특정 조건이 만족되는 동안 코드를 계속해서 반복 실행하는 while 반복문에 대해 학습해 보겠습니다. for 문이 정해진 횟수만큼 반복하는 데 특화되어 있다면, while 문은 특정 상황이 될 때까지 무한히 반복해야 하는 경우에 매우 유용합니다.

1. 파이썬 예제 코드: while 반복문

1.1. 기초 파이썬 예시 코드 (5개)

기초 예제는 while 문의 기본적인 구조와 조건 변화, 그리고 무한 루프 탈출법을 다룹니다.

1. 기본적인 카운터

# 1부터 5까지 숫자를 세며 출력합니다. 조건(count <= 5)이 거짓이 될 때까지 반복합니다.

count = 1
while count <= 5:
    print(f"{count}번째 실행")
    count += 1 # count 값을 1씩 증가시켜 언젠가 조건이 거짓이 되게 만듭니다.

2. 특정 입력이 들어올 때까지 반복

# 사용자가 'exit'를 입력할 때까지 계속해서 입력을 받습니다.

prompt = "메시지를 입력하세요 ('exit'를 입력하면 종료): "
message = ""

while message.lower() != 'exit':
    message = input(prompt)
    if message.lower() != 'exit':
        print(f"입력된 메시지: {message}")

print("프로그램을 종료합니다.")

3. 플래그(flag) 변수 사용하기

# 여러 조건에 의해 종료되어야 할 때 변수를 하나 두어 관리하면 편리합니다.

active = True
while active:
    response = input("계속하시겠습니까? (yes/no): ")
    if response.lower() == 'no':
        active = False
    else:
        print("작업을 진행합니다...")

4. 무한 루프와 break

# while True는 무한 루프를 만들며, 내부에서 break를 만나면 루프를 즉시 탈출합니다.

while True:
    city = input("방문하고 싶은 도시를 입력하세요 (종료하려면 'quit'): ")
    if city.lower() == 'quit':
        break
    print(f"{city}에 가고 싶으시군요!")

5. 리스트 요소 제거하기

# 리스트 안의 특정 값이 모두 사라질 때까지 반복하여 제거합니다.

pets = ['dog', 'cat', 'dog', 'goldfish', 'cat', 'rabbit', 'cat']
print(f"제거 전: {pets}")

while 'cat' in pets:
    pets.remove('cat')

print(f"제거 후: {pets}")

1.2. 응용 파이썬 예시 코드 (5개)

응용 예제는 데이터의 이동, 입력 유효성 검사 등 실질적인 로직 구현을 다룹니다.

1. 리스트 간 데이터 이동하기

# 미완료 목록에서 완료 목록으로 데이터를 하나씩 옮깁니다.

unconfirmed_users = ['홍길동', '구운몽', '인수샘']
confirmed_users = []

while unconfirmed_users:
    current_user = unconfirmed_users.pop()
    print(f"사용자 확인 중: {current_user}")
    confirmed_users.append(current_user)

print("\n--- 확인된 사용자 리스트 ---")
for user in confirmed_users:
    print(user)

2. 숫자를 입력할 때까지 반복하기

# 사용자가 올바른 형식의 숫자를 입력할 때까지 계속 요청합니다.

number = ""
while not number.isdigit():
    number = input("나이를 숫자로만 입력해 주세요: ")
    if not number.isdigit():
        print("잘못된 입력입니다. 다시 입력하세요.")

print(f"당신의 나이는 {number}세입니다.")

3. 간단한 은행 키오스크

# 잔액 내에서 출금을 반복하고 메뉴를 선택하는 구조입니다.

balance = 50000
while True:
    print(f"\n현재 잔액: {balance}원")
    print("1: 입금, 2: 출금, 3: 종료")
    choice = input("원하는 메뉴 번호를 입력하세요: ")

    if choice == '1':
        amount = int(input("입금할 금액: "))
        balance += amount
    elif choice == '2':
        amount = int(input("출금할 금액: "))
        if amount > balance:
            print("잔액이 부족합니다.")
        else:
            balance -= amount
    elif choice == '3':
        print("이용해주셔서 감사합니다.")
        break
    else:
        print("잘못된 메뉴 선택입니다.")

4. continue 사용하기

# 현재 숫자가 짝수일 경우 아래를 건너뛰고 루프의 처음으로 돌아갑니다.

current_number = 0
while current_number < 10:
    current_number += 1
    if current_number % 2 == 0:
        continue
    print(f"홀수 출력: {current_number}")

5. 딕셔너리에 사용자 입력 저장하기

# 반복문을 통해 여러 명의 정보를 딕셔너리에 누적합니다.

responses = {}
polling_active = True

while polling_active:
    name = input("\n성함이 어떻게 되세요? ")
    response = input("어떤 프로그래밍 언어를 좋아하세요? ")
    
    responses[name] = response
    
    repeat = input("다른 분도 참여하시겠습니까? (yes/no) ")
    if repeat.lower() == 'no':
        polling_active = False

print("\n--- 설문 조사 결과 ---")
for name, language in responses.items():
    print(f"{name}님은 {language} 언어를 좋아합니다.")

1.3. 심화 파이썬 예시 코드 (5개)

심화 예제는 알고리즘 구현 및 견고한 예외 처리 기법을 다룹니다.

1. do-while 루프 흉내 내기

# 파이썬에는 `do-while`이 없지만, `while True`와 `break`로 비슷하게 구현할 수 있습니다.

while True:
    response = input("y 또는 n을 입력하세요: ")
    if response.lower() in ('y', 'n'):
        break
print(f"올바른 입력: {response}")

2. 유클리드 호제법으로 최대공약수(GCD) 구하기

# while문을 사용한 고전적인 알고리즘입니다.

a, b = 48, 18
temp_a, temp_b = a, b

while temp_b != 0:
    temp_a, temp_b = temp_b, temp_a % temp_b

print(f"{a}와 {b}의 최대공약수는 {temp_a}입니다.")

3. try-except와 함께 사용하여 견고한 입력 받기

# 숫자가 아닌 입력이 들어와도 프로그램이 죽지 않고 다시 입력받습니다.

while True:
    try:
        age = int(input("나이를 숫자로 입력하세요: "))
        break # 숫자로 변환 성공 시 루프 탈출
    except ValueError:
        print("잘못된 입력입니다. 정수를 입력해 주세요.")

4. 무한 스트림 데이터 처리 흉내 내기

# 특정 조건이 충족될 때까지 누적 합계를 구합니다.

total = 0
count = 0
while total < 100:
    val = int(input(f"현재 합계 {total}. 더할 숫자를 입력하세요: "))
    total += val
    count += 1
print(f"축하합니다! {count}번의 입력 만에 목표치 {total}을(를) 달성했습니다.")

5. 로또 번호 생성 (중복 없이 6개)

# 중복되지 않는 숫자가 6개가 될 때까지 계속 생성합니다.

import random

lotto = []
while len(lotto) < 6:
    num = random.randint(1, 45)
    if num not in lotto:
        lotto.append(num)

lotto.sort()
print(f"이번 주 행운의 번호: {lotto}")
