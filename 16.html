import numpy as np

# 2차원 배열 생성 (행렬)
matrix = np.array([[1, 2, 3], [4, 5, 6]])

print(f"차원(ndim): {matrix.ndim}")

브로드캐스팅 (Broadcasting)

크기가 다른 배열 간의 연산을 가능하게 하는 강력한 기능입니다. 작은 배열이 큰 배열의 형태에 맞춰 확장되어 연산됩니다.

# (2, 3) 배열 + 스칼라(10)
arr = np.array([[1, 2, 3], [4, 5, 6]])
result = arr + 10 

# 모든 요소에 10이 더해짐
# [[11, 12, 13], [14, 15, 16]]
💡 연산 성능: Numpy 연산은 내부적으로 C로 구현된 루프를 사용하기 때문에, 일반적인 파이썬 for문보다 수백 배 이상 빠릅니다.

고급 연산 기법

조건에 따른 값 할당(np.where)과 수학적 행렬 곱셈(@)을 다룹니다.

# 1. 조건부 값 할당 (if-else의 벡터화)
scores = np.array([55, 92, 78, 40])
result = np.where(scores >= 60, "Pass", "Fail")
print(result) # ['Fail' 'Pass' 'Pass' 'Fail']

# 2. 행렬 곱셈 (Dot Product)
mat1 = np.array([[1, 2], [3, 4]])
mat2 = np.array([[5, 6], [7, 8]])
dot_res = mat1 @ mat2 # Python 3.5 이상 권장 방식
