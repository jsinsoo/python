4차시: 파이썬 연산자 학습 패키지

값들을 계산하고, 비교하고, 논리적인 판단을 내리는 연산자를 학습합니다.

1. 기초 파이썬 예제 [cite: 172]
가장 빈번하게 사용되는 기본적인 연산자들을 다룹니다. [cite: 172]

1.1 산술 연산자 (Arithmetic)

숫자 계산에 사용되는 기본적인 연산자입니다. [cite: 173]

a, b = 10, 3
print(f"더하기: {a + b}")    # 13
print(f"나눈 몫: {a // b}")   # 3
print(f"나머지: {a % b}")     # 1
print(f"거듭제곱: 2 ** 4 = {2**4}") # 16 [cite: 174]

1.2 비교 및 논리 연산자

값을 비교하거나 여러 조건을 조합할 때 사용합니다. [cite: 175, 176]

# 비교: True 또는 False 반환
print(10 > 5)  # True [cite: 176]

# 논리: AND, OR, NOT
a, b = True, False
print(a and b) # False (둘 다 참이어야 참)
print(a or b)  # True (하나라도 참이면 참) [cite: 177]
2. 응용 및 심화 예제 [cite: 179, 190]
코드의 효율성을 높이는 특수 연산자와 저수준 비트 연산을 다룹니다. [cite: 179, 190]

2.1 멤버십 및 삼항 연산자

리스트 내 값 존재 여부 확인 및 조건문을 한 줄로 표현합니다. [cite: 180, 185]

fruits = ["사과", "바나나"]
print("사과" in fruits) # True (멤버십 연산자) [cite: 181]

score = 85
result = "합격" if score >= 80 else "불합격" # 삼항 연산자 [cite: 185]

2.2 비트 연산 및 왈러스 연산자

데이터의 비트 단위 제어와 Python 3.8의 할당 표현식입니다. [cite: 191, 198]

# 비트 AND 연산 (&)
print(5 & 3) # 1 (0101 & 0011 = 0001) [cite: 192]

# 왈러스 연산자 (:=) : 할당과 조건 검사를 동시에
while (text := input("입력: ")) != "quit":
    print(f"입력값: {text}") [cite: 198]
