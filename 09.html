9차시: 리스트(List) 학습 패키지

안녕하세요! 파이썬에서 가장 중요하고 활용도 높은 자료구조인 리스트(List)에 대해 학습해 보겠습니다. 여러 개의 값을 순서대로 저장하고, 자유롭게 수정, 추가, 삭제할 수 있는 리스트의 모든 것을 예제 코드와 퀴즈를 통해 마스터해 보세요.

1. 파이썬 예제 코드: 리스트

1.1. 기초 파이썬 예시 코드 (5개)

기초 예제는 리스트를 만들고, 특정 요소에 접근하며, 값을 추가하고 수정하는 기본적인 방법을 다룹니다.

1. 리스트 생성 및 요소 접근

# 다양한 자료형을 담을 수 있는 리스트를 생성합니다. 인덱스는 0부터 시작합니다.

my_list = [10, "hello", True, 3.14]

print("첫 번째 요소:", my_list[0])      # 10
print("두 번째 요소:", my_list[1])      # "hello"
# 음수 인덱스는 뒤에서부터 접근합니다.
print("마지막 요소:", my_list[-1])     # 3.14

2. 리스트 요소 변경하기

# 리스트는 '변경 가능한(mutable)' 자료형이므로 인덱스를 통해 값을 바꿀 수 있습니다.

numbers = [1, 2, 3, 4, 5]
print("변경 전:", numbers)

numbers[2] = 30 # 3번째 요소(인덱스 2)를 30으로 변경
print("변경 후:", numbers)

3. 리스트에 요소 추가하기 (append, insert)

# append는 맨 뒤에, insert는 원하는 위치에 요소를 추가합니다.

fruits = ["사과", "바나나"]
fruits.append("딸기")     # 맨 뒤에 추가
print("append 후:", fruits)

fruits.insert(1, "포도")  # 인덱스 1 위치에 추가
print("insert 후:", fruits)

4. 리스트 요소 삭제하기 (remove, pop, del)

# 값으로 삭제하거나 인덱스로 삭제할 수 있습니다.

items = ["A", "B", "C", "D", "E"]

items.remove("B")       # 값 "B"를 찾아 삭제
print("remove 후:", items)

popped_item = items.pop() # 마지막 요소를 꺼내고 삭제
print(f"pop된 요소: {popped_item}, 남은 리스트: {items}")

del items[0]            # 인덱스 0 요소를 삭제
print("del 후:", items)

5. 리스트 슬라이싱 (Slicing)

# 리스트의 일부분을 잘라내어 새로운 리스트를 만듭니다. [시작:끝]에서 끝은 포함되지 않습니다.

chars = ['a', 'b', 'c', 'd', 'e']
print("chars[1:4]:", chars[1:4]) # 인덱스 1부터 3까지 ('b', 'c', 'd')
print("chars[:3]:", chars[:3])   # 처음부터 인덱스 2까지
print("chars[2:]:", chars[2:])   # 인덱스 2부터 끝까지

1.2. 응용 파이썬 예시 코드 (5개)

응용 예제는 리스트 정렬, 탐색, 변환 등 실질적인 데이터 처리를 다룹니다.

1. 리스트 정렬 (sort, sorted)

# sort()는 원본을 정렬하고, sorted()는 정렬된 새 리스트를 반환합니다.

nums = [5, 2, 9, 1, 7]

# 오름차순 정렬
nums.sort()
print("오름차순 정렬:", nums)

# 내림차순 정렬
nums.sort(reverse=True)
print("내림차순 정렬:", nums)

2. 리스트 요소의 인덱스 찾기 및 개수 세기

# index()로 위치를 찾고, count()로 특정 요소의 개수를 셉니다.

colors = ["red", "blue", "green", "blue", "yellow"]

print("blue의 인덱스:", colors.index("blue")) # 처음 발견된 위치
print("blue의 개수:", colors.count("blue"))

3. 리스트 뒤집기 및 확장

# reverse()로 순서를 반전시키고, extend()로 다른 리스트를 합칩니다.

list1 = [1, 2, 3]
list2 = [4, 5]

list1.reverse()
print("뒤집힌 list1:", list1)

list1.extend(list2) # list1 += list2 와 유사
print("확장된 list1:", list1)

4. 문자열과 리스트 사이의 변환 (split, join)

# 문자열을 리스트로 쪼개고, 리스트를 문자열로 합칩니다.

text = "인수샘,홍길동,구운몽"
name_list = text.split(",") # 쉼표를 기준으로 리스트 생성
print("리스트로 변환:", name_list)

joined_text = " - ".join(name_list) # 리스트 요소를 하이픈으로 연결
print("문자열로 합치기:", joined_text)

5. 리스트 요소 중 최댓값, 최솟값, 합계

# 내장 함수를 사용하여 리스트를 요약합니다.

scores = [85, 92, 78, 95, 88]

print("최고 점수:", max(scores))
print("최저 점수:", min(scores))
print("점수 합계:", sum(scores))
print("평균 점수:", sum(scores) / len(scores))

1.3. 심화 파이썬 예시 코드 (3개)

심화 예제는 중첩 리스트와 복사(Shallow vs Deep Copy) 개념을 다룹니다.

1. 리스트 컴프리헨션 (List Comprehension)

# 기존 리스트를 바탕으로 새로운 리스트를 간결하게 생성합니다.

# 1부터 10까지 숫자 중 짝수의 제곱만 포함하는 리스트 생성
even_squares = [x**2 for x in range(1, 11) if x % 2 == 0]
print("짝수 제곱 리스트:", even_squares)

2. 중첩 리스트 (Nested List)와 2차원 배열

# 리스트 내부에 또 다른 리스트를 포함할 수 있습니다.

matrix = [
    [1, 2, 3],
    [4, 5, 6],
    [7, 8, 9]
]
# 2행 3열의 요소 (인덱스는 0부터 시작하므로 1행 2열 접근)
print("2행 3열의 요소:", matrix[1][2]) # 6

# 중첩 for문으로 모든 요소 출력
print("--- 행렬 전체 출력 ---")
for row in matrix:
    for element in row:
        print(element, end=" ")
    print() # 줄바꿈

3. 리스트 복사: 얕은 복사 vs 깊은 복사

# 중첩 리스트의 경우 일반적인 copy()는 내부 리스트까지 복사하지 못합니다.

import copy

# 얕은 복사: 내부의 리스트는 같은 객체를 공유함
list_a = [1, [10, 20]]
list_b = list_a.copy() 

list_b[0] = 99
list_b[1].append(30) # 내부 리스트에 요소 추가

print("얕은 복사 후 원본(a):", list_a) # 내부 리스트가 수정되어 영향을 받음
print("얕은 복사 후 사본(b):", list_b)
print("-" * 20)

# 깊은 복사: 내부의 리스트까지 모두 완전히 새로 복사함
list_c = [1, [10, 20]]
list_d = copy.deepcopy(list_c)

list_d[0] = 99
list_d[1].append(30)

print("깊은 복사 후 원본(c):", list_c) # 원본 보존됨
print("깊은 복사 후 사본(d):", list_d)
