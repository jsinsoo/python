9차시: 리스트(List) 마스터

여러 개의 데이터를 하나로 묶어 자유롭게 관리하는 자료구조의 핵심을 배웁니다.

STEP 1. 기초 다지기

리스트를 생성하고 인덱스를 통해 데이터에 접근하는 법을 학습합니다.

1.1 생성 및 인덱싱

파이썬의 인덱스는 0부터 시작하며, 마이너스(-) 인덱스는 뒤에서부터 거꾸로 셉니다.
# 다양한 자료형을 담는 리스트
my_list = [10, "hello", True, 3.14]

print(my_list[0])   # 10
print(my_list[-1])  # 3.14 (마지막 요소)

1.2 리스트 요소 변경 (Mutable)

리스트는 값을 직접 바꿀 수 있는 '변경 가능(Mutable)' 자료형입니다.
numbers = [1, 2, 3, 4, 5]
numbers[2] = 30 # 3번째 요소를 30으로 교체
print(numbers)  # [1, 2, 30, 4, 5]
STEP 2. 리스트 조작법

데이터를 추가, 삭제하고 정렬하는 등 실전에서 자주 쓰이는 메서드를 다룹니다.

2.1 요소 추가 및 삭제

fruits = ["사과", "바나나"]
fruits.append("포도")      # 맨 뒤에 추가
fruits.insert(1, "딸기")   # 원하는 위치(인덱스 1)에 삽입
fruits.remove("바나나")    # 값으로 삭제
last = fruits.pop()        # 마지막 요소 꺼내기 및 삭제

2.2 슬라이싱 (Slicing)

[시작:끝] 형식을 사용하며, 끝 인덱스는 포함되지 않음에 유의하세요.
nums = [0, 1, 2, 3, 4, 5, 6]
print(nums[1:4])  # [1, 2, 3]
print(nums[:3])   # [0, 1, 2] (처음부터 3 전까지)
STEP 3. 심화 개념

2차원 리스트와 주의해야 할 리스트 복사 개념을 다룹니다.

3.1 2차원 리스트 (중첩 리스트)

matrix = [
    [1, 2, 3],
    [4, 5, 6]
]
print(matrix[1][2]) # 2행 3열의 값인 6 출력

3.2 얕은 복사 vs 깊은 복사

⚠️ 주의! 단순히 list_b = list_a라고 적으면 두 변수는 같은 데이터를 가리키게 되어 하나를 수정하면 둘 다 바뀝니다. 
import copy

# 깊은 복사: 완전히 독립적인 새로운 리스트 생성
original = [1, [10, 20]]
deep_copy = copy.deepcopy(original)

deep_copy[1].append(30)
print(f"원본: {original}")    # [1, [10, 20]] (유지됨)
print(f"사본: {deep_copy}")   # [1, [10, 20, 30]]
