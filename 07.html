7차시: for 반복문 학습 패키지

안녕하세요! 프로그래밍의 가장 강력한 기능 중 하나인 '반복'을 수행하는 for 반복문에 대해 학습해 보겠습니다. 정해진 횟수나 범위만큼 코드를 반복 실행하는 for 문의 다양한 활용법을 예제 코드와 퀴즈로 익혀보세요.

1. 파이썬 예제 코드: for 반복문

1.1. 기초 파이썬 예시 코드 (5개)

기초 예제는 for 문의 가장 기본적인 사용법인 range() 함수와 시퀀스 순회에 대해 다룹니다.

1. range()로 정해진 횟수만큼 반복

# 0부터 4까지, 총 5번 반복합니다.

for i in range(5):
    print(f"{i+1}번째 안녕, 파이썬!")

2. 리스트(List)의 모든 요소 순회하기

# 리스트에 들어있는 모든 항목을 하나씩 꺼내어 출력합니다.

fruits = ["사과", "바나나", "딸기", "오렌지"]
for fruit in fruits:
    print(f"내가 좋아하는 과일: {fruit}")

3. 문자열(String)의 모든 문자 순회하기

# 문자열을 구성하는 모든 글자를 하나씩 순서대로 출력합니다.

for char in "Hello":
    print(char)

4. range()에 시작과 끝 값 지정하기

# 1부터 5까지의 숫자를 더합니다. (range(start, end)에서 end는 포함되지 않음)

total = 0
for i in range(1, 6):
    total += i
print(f"1부터 5까지의 합: {total}")

5. range()에 간격(step) 지정하기

# 1부터 10 사이의 홀수만 출력합니다.

for i in range(1, 11, 2):
    print(f"홀수: {i}")

1.2. 응용 파이썬 예시 코드 (5개)

응용 예제는 반복문 안에 조건문을 넣거나 여러 리스트를 동시에 다루는 방법을 다룹니다.

1. 중첩 반복문 (Nested For Loop)

# 반복문 안에 또 다른 반복문을 사용하여 구구단을 출력합니다.

for i in range(2, 4):  # 2단부터 3단까지
    print(f"--- {i}단 ---")
    for j in range(1, 10):
        print(f"{i} x {j} = {i * j}")

2. enumerate()로 인덱스와 값 함께 가져오기

# 리스트의 순서(번호)가 필요할 때 사용합니다.

members = ["인수샘", "홍길동", "구운몽"]
for idx, name in enumerate(members):
    print(f"{idx + 1}번 회원: {name}")

3. zip()으로 여러 리스트 동시 순회

# 두 개 이상의 리스트 요소를 짝지어 가져올 때 유용합니다.

names = ["홍길동", "구운몽"]
scores = [85, 92]
for name, score in zip(names, scores):
    print(f"{name}의 점수: {score}점")

4. 딕셔너리(Dictionary) 순회

# 키(key)와 값(value)을 동시에 꺼내어 처리합니다.

capitals = {"한국": "서울", "일본": "도쿄", "미국": "워싱턴"}
for country, city in capitals.items():
    print(f"{country}의 수도는 {city}입니다.")

5. 반복문 안에서 조건문 활용 (짝수만 추출)

# 조건에 맞는 요소만 선택하여 새로운 리스트를 만듭니다.

original = [1, 2, 3, 4, 5, 6, 7, 8]
doubled_evens = []
for n in original:
    if n % 2 == 0:
        doubled_evens.append(n * 2)
print(f"원본 리스트의 짝수를 2배하여 만든 리스트: {doubled_evens}")

1.3. 심화 파이썬 예시 코드 (5개)

심화 예제는 for 문을 더 간결하고 효율적으로 사용하는 고급 기법들을 다룹니다.

1. 리스트 컴프리헨션 (List Comprehension)

# 응용 예제 5번을 한 줄로 압축한 "파이썬스러운" 코드입니다.

original = [1, 2, 3, 4, 5, 6, 7, 8]
doubled_evens = [n * 2 for n in original if n % 2 == 0]
print(f"리스트 컴프리헨션 결과: {doubled_evens}")

2. break로 반복문 탈출하기

# 특정 조건을 만족했을 때 반복을 즉시 종료합니다.

numbers = [3, 9, 2, 15, 7, 8]
# 처음으로 10보다 큰 숫자를 찾으면 중단
for num in numbers:
    if num > 10:
        print(f"발견한 첫 10 초과 숫자: {num}")
        break

3. continue로 다음 반복 진행하기

# 특정 조건에서 아래 코드를 건너뛰고 다음 반복으로 넘어갑니다.

for i in range(1, 6):
    if i == 3:
        continue # 3인 경우 출력하지 않고 건너뜀
    print(f"숫자: {i}")

4. for-else 구문

# 반복문이 break 없이 끝까지 완주했을 때 실행됩니다.

for i in range(5):
    # i가 10인 경우는 없으므로 break가 실행되지 않음
    if i == 10:
        break
else:
    print("반복문이 중단 없이 정상 종료되었습니다.")

5. 다중 리스트 컴프리헨션

# 이중 반복문을 한 줄의 리스트 컴프리헨션으로 구현합니다.

# (x, y) 형태의 좌표 리스트 만들기
coords = [(x, y) for x in range(2) for y in range(2)]
print(f"생성된 좌표: {coords}")
