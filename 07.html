7차시: for 반복문 마스터

동일한 작업을 효율적으로 수행하는 컴퓨터의 핵심 능력을 배웁니다.

1. 기초 반복문 (range & 시퀀스)

정해진 횟수만큼 실행하거나, 데이터 꾸러미(리스트, 문자열)를 하나씩 꺼내어 처리합니다.

[Image of for loop flow diagram in python]

1.1 range() 함수 활용

range(5)는 0부터 4까지 총 5번의 숫자를 생성합니다.
# 0부터 4까지 반복
for i in range(5):
    print(f"{i+1}번째 안녕, 파이썬!")

1.2 리스트 요소 순회

fruits = ["사과", "바나나", "딸기", "오렌지"]
for fruit in fruits:
    print(f"내가 좋아하는 과일: {fruit}")
2. 응용 반복문 (데이터 가공)

반복문 내부에 조건문을 넣어 특정 데이터를 필터링하거나 누적 계산을 수행합니다.

2.1 조건문과 결합 (짝수만 처리)

리스트에서 특정 조건(% 2 == 0)을 만족하는 요소만 골라낼 수 있습니다.
original = [1, 2, 3, 4, 5, 6, 7, 8]
doubled_evens = []
for n in original:
    if n % 2 == 0:
        doubled_evens.append(n * 2)
print(f"짝수를 2배한 결과: {doubled_evens}")
3. 심화 기법 (효율적인 코드)

파이썬스러운(Pythonic) 코드 작성법과 반복문 제어 기법입니다.

3.1 리스트 컴프리헨션

여러 줄의 코드를 한 줄로 압축하여 성능과 가독성을 모두 잡습니다.
# [표현식 for 항목 in 반복가능객체 if 조건]
original = [1, 2, 3, 4, 5, 6, 7, 8]
result = [n * 2 for n in original if n % 2 == 0]
print(f"한 줄 코드 결과: {result}")

3.2 break를 이용한 조기 종료

numbers = [3, 9, 2, 15, 7, 8]
# 처음으로 발견되는 5보다 큰 짝수를 찾으면 즉시 멈춤
for num in numbers:
    if num > 5 and num % 2 == 0:
        print(f"찾았다: {num}")
        break
