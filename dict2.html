파이썬 딕셔너리(Dictionary) 학습 요소 10가지

키와 값의 구조부터 조회, 추가, 수정, 삭제, 반복, 병합까지 한 번에 정리

목차

1. 딕셔너리 생성 2. 값 조회 3. get() 사용 4. 데이터 추가 5. 데이터 수정 6. 데이터 삭제 7. keys() 8. values() 9. items() 10. 딕셔너리 병합 정리표

1. 딕셔너리 생성하기

딕셔너리는 키(Key)와 값(Value)을 한 쌍으로 저장하는 자료형입니다.
student = {
    "이름": "민수",
    "나이": 17,
    "학교": "과학고"
}

print(student)
출력:
{'이름': '민수', '나이': 17, '학교': '과학고'}

2. 키를 이용하여 값 조회하기

딕셔너리는 순서 번호가 아니라 키를 이용해 값을 가져옵니다.
student = {"이름": "민수", "나이": 17}

print(student["이름"])
출력:
민수

3. get() 메서드 사용하기

get()은 존재하지 않는 키를 조회해도 오류가 발생하지 않고 None을 반환합니다.
student = {"이름": "민수"}

print(student.get("이름"))
print(student.get("주소"))
출력:
민수
None

4. 데이터 추가하기

새로운 키에 값을 대입하면 딕셔너리에 데이터가 추가됩니다.
student = {"이름": "민수"}

student["학년"] = 2

print(student)
출력:
{'이름': '민수', '학년': 2}

5. 데이터 수정하기

이미 존재하는 키에 새로운 값을 대입하면 기존 값이 수정됩니다.
student = {"이름": "민수", "나이": 17}

student["나이"] = 18

print(student)
출력:
{'이름': '민수', '나이': 18}

6. 데이터 삭제하기

del 또는 pop()을 사용하여 특정 키와 값을 삭제할 수 있습니다.
student = {"이름": "민수", "나이": 17}

del student["나이"]

print(student)
출력:
{'이름': '민수'}

7. keys() 사용하기

keys()는 딕셔너리에 들어 있는 모든 키를 가져옵니다.
student = {"이름": "민수", "나이": 17, "학교": "과학고"}

print(student.keys())
출력:
dict_keys(['이름', '나이', '학교'])

8. values() 사용하기

values()는 딕셔너리에 들어 있는 모든 값을 가져옵니다.
student = {"이름": "민수", "나이": 17, "학교": "과학고"}

print(student.values())
출력:
dict_values(['민수', 17, '과학고'])

9. items() 사용하기

items()는 키와 값을 동시에 가져올 때 사용합니다.
student = {"이름": "민수", "나이": 17, "학교": "과학고"}

for key, value in student.items():
    print(key, value)
출력:
이름 민수
나이 17
학교 과학고

10. 딕셔너리 병합하기

파이썬 3.9 이상에서는 | 연산자를 사용하여 두 딕셔너리를 병합할 수 있습니다.
d1 = {"a": 1}
d2 = {"b": 2}

d3 = d1 | d2

print(d3)
출력:
{'a': 1, 'b': 2}

2조회키를 이용해 값 가져오기student["이름"]
3get()오류 없이 안전하게 조회student.get("주소")

7keys()모든 키 확인student.keys()
8values()모든 값 확인student.values()

간단 확인 문제 보기

문제: student = {'이름':'민수', '나이':17}에서 나이를 18로 바꾸는 코드는?

정답: student['나이'] = 18

해설: 이미 존재하는 키에 새로운 값을 대입하면 값이 수정됩니다.
