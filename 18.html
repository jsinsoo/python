18차시: 파이썬 프로젝트 마스터

기초 문법부터 라이브러리 활용까지, 나만의 실전 프로그램을 만들어봅니다.

LEVEL 1: BASIC

숫자 맞추기 게임

  • random 모듈을 활용한 난수 생성
  • while 루프와 조건문을 통한 반복 피드백
  • try-except를 이용한 예외 처리(입력 오류 방지)
import random
secret = random.randint(1, 100)

while True:
    guess = int(input("1~100 사이 숫자 입력: "))
    if guess < secret: print("더 크게!")
    elif guess > secret: print("더 작게!")
    else:
        print("정답입니다!"); break
LEVEL 1: BASIC

심플 함수 계산기

  • 함수 정의를 통한 로직 분리
  • 연산자별 조건 처리
  • 사용자 입력 데이터의 연산 수행
def calculate(x, op, y):
    if op == '+': return x + y
    elif op == '-': return x - y
    elif op == '*': return x * y
    elif op == '/': return x / y if y != 0 else "Error"

# 실행 로직 생략
LEVEL 2: EXTERNAL LIB

유튜브 영상 다운로더

필수 설치: pip install pytube
  • 외부 라이브러리(pytube) 활용
  • 영상 URL 분석 및 최고 화질 추출
  • 로컬 PC로 파일 스트리밍 다운로드
from pytube import YouTube

url = input("유튜브 URL: ")
yt = YouTube(url)
stream = yt.streams.get_highest_resolution()
stream.download()
print(f"[{yt.title}] 다운로드 완료!")
LEVEL 3: ADVANCED

이미지 텍스트 추출 (OCR)

설치: pip install pytesseract pillow
  • Pillow를 이용한 이미지 로딩
  • Tesseract 엔진 기반 문자 인식
  • 비정형 데이터(이미지)의 텍스트화
from PIL import Image
import pytesseract

img = Image.open('sample_text.png')
text = pytesseract.image_to_string(img, lang='kor')
print("인식 결과:", text)
