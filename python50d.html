예제 175. 재귀 함수: 팩토리얼

def factorial(number):
    if number == 1:
        return 1
    return number * factorial(number - 1)

print(factorial(5))
실행 결과
120
설명
재귀 함수는 함수가 자기 자신을 다시 호출하는 방식입니다. 반드시 종료 조건이 필요합니다.

3. 람다 함수와 함수형 도구 — 예제 176~185

예제 178. 조건이 있는 람다 함수

홀수
설명
람다 함수 안에서도 삼항 연산자를 이용해 간단한 조건 처리를 할 수 있습니다.

예제 179. map으로 제곱 리스트 만들기

numbers = [1, 2, 3, 4, 5]

result = list(map(lambda number: number ** 2, numbers))

print(result)

[1, 4, 9, 16, 25]
설명
map()은 모든 요소에 같은 함수를 적용합니다. 결과를 리스트로 보려면 list()로 변환합니다.

예제 180. map으로 문자열 숫자 변환

text_numbers = ["10", "20", "30"]

print(numbers)

[10, 20, 30]
설명
map(int, 리스트)는 문자열 숫자를 정수로 한꺼번에 변환할 때 자주 사용합니다.

예제 181. filter로 짝수만 추출

numbers = [1, 2, 3, 4, 5, 6]

even_numbers = list(filter(lambda number: number % 2 == 0, numbers))

print(even_numbers)
실행 결과
[2, 4, 6]
설명
filter()는 조건이 True인 요소만 남깁니다.

예제 182. filter로 긴 단어 찾기

words = ["봄", "파이썬", "프로그래밍", "코드"]

long_words = list(filter(lambda word: len(word) >= 3, words))

실행 결과
['파이썬', '프로그래밍']
설명
filter와 lambda를 사용하면 조건에 맞는 데이터만 간단히 추출할 수 있습니다.

예제 183. sorted와 lambda

students = [
    {"name": "민수", "score": 80},
    {"name": "지민", "score": 95},
    {"name": "서연", "score": 88}
]

result = sorted(students, key=lambda student: student["score"])

print(result)
실행 결과
[{'name': '민수', 'score': 80}, {'name': '서연', 'score': 88}, {'name': '지민', 'score': 95}]
설명
key에 람다 함수를 넣으면 원하는 기준으로 정렬할 수 있습니다.

예제 184. 문자열 길이 기준 정렬

words = ["봄", "파이썬", "코딩", "프로그래밍"]

예제 200. 함수와 내장 함수 종합

def analyze_scores(scores):
    return {
        "합계": sum(scores),
        "평균": sum(scores) / len(scores),
        "최고점": max(scores),
        "최저점": min(scores)
    }

scores = [85, 92, 78, 100]
result = analyze_scores(scores)

for key, value in result.items():
    print(key, ":", value)
실행 결과
합계 : 355
평균 : 88.75
최고점 : 100
최저점 : 78
설명
함수, 리스트, 딕셔너리, 반복문, 내장 함수를 결합하면 실제 데이터 처리 프로그램을 만들 수 있습니다.

4편 핵심 요약

  • def로 함수를 정의하고, 필요한 기능을 재사용할 수 있습니다.
  • return은 함수의 결과를 호출한 위치로 돌려줍니다.
  • 기본값, 키워드 인자, *args, **kwargs를 사용하면 유연한 함수를 만들 수 있습니다.
  • lambda, map(), filter(), sorted()는 데이터를 간결하게 처리할 때 유용합니다.
