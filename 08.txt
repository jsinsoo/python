8차시: while 반복문 학습 패키지
안녕하세요! 이번 시간에는 특정 조건이 만족되는 동안 코드를 계속해서 반복 실행하는 while 반복문에 대해 학습해 보겠습니다. for 문이 정해진 횟수만큼 반복하는 데 특화되어 있다면, while 문은 특정 상황이 될 때까지 무한히 반복해야 하는 경우에 매우 유용합니다.

1. 파이썬 예제 코드: while 반복문
1.1. 기초 파이썬 예시 코드 (5개)
기초 예제는 while 문의 기본적인 구조와 조건 변화, 그리고 무한 루프 탈출법을 다룹니다.

1. 기본적인 카운터

# 1부터 5까지 숫자를 세며 출력합니다.
# 조건(count <= 5)이 거짓이 될 때까지 반복합니다.
count = 1
while count <= 5:
    print(f"{count}번째 실행")
    count += 1 # count 값을 1씩 증가시켜 언젠가 조건이 거짓이 되게 만듭니다.
2. 특정 입력이 들어올 때까지 반복

# 사용자가 'exit'를 입력할 때까지 계속해서 입력을 받습니다.
prompt = "메시지를 입력하세요 ('exit'를 입력하면 종료): "
message = ""

while message.lower() != 'exit':
    message = input(prompt)
    if message.lower() != 'exit':
        print(f"입력된 메시지: {message}")

print("프로그램을 종료합니다.")
3. 플래그(flag) 변수 사용하기

# 반복을 계속할지 결정하는 '깃발' 역할을 하는 변수를 사용합니다.
active = True
count = 0
while active:
    print(f"현재 카운트: {count}")
    count += 1
    if count >= 3:
        active = False # 플래그를 내려서 다음 반복 때 루프가 종료되도록 합니다.
4. while True와 break를 이용한 무한 루프 탈출

# 가장 흔하게 사용되는 무한 루프 패턴입니다.
# 루프 안에서 특정 조건을 만족하면 `break`를 이용해 탈출합니다.
while True:
    answer = input("계속할까요? (y/n): ")
    if answer.lower() == 'n':
        break # break를 만나면 즉시 while 루프를 탈출합니다.
print("반복이 종료되었습니다.")
5. 카운트다운

# 변수 값을 감소시키면서 반복할 수도 있습니다.
n = 5
while n > 0:
    print(n)
    n -= 1
print("발사!")
1.2. 응용 파이썬 예시 코드 (5개)
응용 예제는 while 문을 리스트 처리, 메뉴 구성, else, continue 등과 함께 사용하는 방법을 다룹니다.

1. 리스트의 모든 요소 처리하기 (pop)

# 리스트가 비어있지 않은 동안(True로 간주됨) 마지막 요소를 계속 꺼냅니다.
tasks = ["코딩", "운동", "독서"]
while tasks:
    task = tasks.pop() # 리스트의 마지막 요소를 꺼내고 리스트에서 제거
    print(f"처리할 작업: {task}")
print("모든 작업을 완료했습니다!")
2. continue로 조건에 맞지 않으면 건너뛰기

# 1부터 10까지 숫자 중 홀수만 출력합니다.
num = 0
while num < 10:
    num += 1
    if num % 2 == 0: # 숫자가 짝수이면
        continue     # 아래의 print를 실행하지 않고 다음 반복으로 넘어갑니다.
    print(num, end=" ")
3. while-else 문

# while 루프가 break로 중단되지 않고, 조건이 거짓이 되어 정상 종료되었을 때만 else 블록이 실행됩니다.
count = 0
while count < 3:
    print(f"{count+1}번째 시도...")
    count += 1
else:
    # break 없이 루프가 끝났으므로 이 부분이 실행됩니다.
    print("루프가 정상적으로 종료되었습니다.")
4. 간단한 숫자 맞추기 게임

# while, if, break를 모두 사용한 간단한 프로그램입니다.
import random
secret_number = random.randint(1, 20)
print("1부터 20 사이의 숫자를 맞춰보세요!")

while True:
    guess = int(input("추측한 숫자는? "))
    if guess < secret_number:
        print("더 큰 숫자입니다!")
    elif guess > secret_number:
        print("더 작은 숫자입니다!")
    else:
        print(f"정답입니다! 숫자는 {secret_number}였습니다.")
        break
5. 간단한 메뉴 선택 프로그램

# 사용자가 종료를 선택할 때까지 계속해서 메뉴를 보여줍니다.
while True:
    print("\n--- 메뉴 ---")
    print("1. 인사하기")
    print("2. 노래하기")
    print("3. 종료하기")
    choice = input("선택: ")

    if choice == '1':
        print("안녕하세요!")
    elif choice == '2':
        print("랄랄라~")
    elif choice == '3':
        print("프로그램을 종료합니다.")
        break
    else:
        print("잘못된 선택입니다. 다시 입력해주세요.")
1.3. 심화 파이썬 예시 코드 (5개)
심화 예제는 while 문을 알고리즘이나 더 복잡한 제어 구조에 활용하는 방법을 다룹니다.

1. do-while 루프 흉내 내기

# 파이썬에는 `do-while`이 없지만, `while True`와 `break`로 비슷하게 구현할 수 있습니다.
# 먼저 코드 블록을 한 번 실행하고, 그 다음 조건을 검사합니다.
while True:
    # 먼저 실행하고 싶은 코드
    response = input("y 또는 n을 입력하세요: ")
    
    # 마지막에 조건 검사 후 탈출
    if response.lower() in ('y', 'n'):
        break
print(f"올바른 입력: {response}")
2. 유클리드 호제법으로 최대공약수(GCD) 구하기

# while문을 사용한 고전적인 알고리즘입니다.
a = 48
b = 18

temp_a, temp_b = a, b
while temp_b != 0:
    temp_a, temp_b = temp_b, temp_a % temp_b

print(f"{a}와 {b}의 최대공약수는 {temp_a}입니다.")
3. try-except와 함께 사용하여 견고한 입력 받기

# 사용자가 숫자를 입력할 때까지 계속해서 입력을 요청합니다.
while True:
    try:
        age = int(input("나이를 숫자로 입력하세요: "))
        break # 숫자로 변환 성공 시 루프 탈출
    except ValueError:
        # int() 변환 실패 시 ValueError 발생
        print("잘못된 입력입니다. 반드시 숫자로 입력해주세요.")

print(f"입력된 나이: {age}")
4. 제너레이터(Generator)와 함께 사용하기

# 대용량 데이터를 처리할 때 메모리를 효율적으로 사용할 수 있습니다.
def fibonacci_generator():
    a, b = 0, 1
    while True:
        yield a
        a, b = b, a + b

fib = fibonacci_generator()
count = 0
while count < 10:
    print(next(fib), end=" ")
    count += 1
5. 간단한 게임 루프

# 게임 캐릭터가 특정 위치에 도달할 때까지 계속 움직입니다.
import time

player_pos = 0
target_pos = 10
speed = 2

while player_pos < target_pos:
    print(f"플레이어 위치: {player_pos}")
    player_pos += speed
    time.sleep(0.5) # 0.5초 대기
    
    if player_pos >= target_pos:
        player_pos = target_pos # 목표 지점을 넘어가지 않도록 보정

print(f"목표({target_pos})에 도달했습니다!")
