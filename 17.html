pip install matplotlib
BASIC PLOT

기본 그래프와 환경 설정

가장 기본이 되는 선 그래프(Line Plot)와 한글 폰트 설정 방법을 학습합니다.

그래프의 주요 구성 요소: 제목, 축 레이블, 그리드
⚠️ 한글 깨짐 방지: Matplotlib은 기본적으로 한글을 지원하지 않으므로 rcParams를 통해 나눔고딕이나 맑은 고딕 등을 설정해야 합니다. 
import matplotlib.pyplot as plt

# 데이터 준비
x = [1, 2, 3, 4, 5]
y = [10, 25, 18, 32, 28]

# 그래프 설정
plt.plot(x, y, marker='o', linestyle='--', color='b', label='성적 추이')
plt.title('월별 성적 변화', fontsize=15)
plt.xlabel('월')
plt.ylabel('점수')
plt.legend() # 범례 표시
plt.grid(True)
plt.show()
VISUAL GALLERY

데이터 특성에 따른 그래프 선택

비교에는 막대 그래프, 분포에는 히스토그램, 관계에는 산점도를 사용합니다.

# 1. 막대 그래프 (Bar Chart) - 범주별 비교
categories = ['A', 'B', 'C']
values = [100, 150, 120]
plt.bar(categories, values, color=['red', 'green', 'blue'])

# 2. 산점도 (Scatter Plot) - 변수 간 상관관계
import numpy as np
x = np.random.rand(50)
y = np.random.rand(50)
plt.scatter(x, y, alpha=0.5, s=100) # s는 점의 크기
ADVANCED VIZ

주석 추가와 3D 시각화

특정 지점을 강조하는 Annotation과 입체적인 3D Plot을 다룹니다.

# 주석(Annotation) 추가하기
plt.plot([1, 2, 3], [10, 20, 30])
plt.annotate('중요 지점', xy=(2, 20), xytext=(2.5, 25),
             arrowprops=dict(facecolor='black', shrink=0.05))

# 서브플롯 (여러 그래프 동시에 그리기)
fig, axes = plt.subplots(1, 2, figsize=(10, 4))
axes[0].plot([1,2,3], [1,4,9])
axes[1].bar(['X', 'Y'], [10, 20])
