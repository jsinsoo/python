17차시: Matplotlib 데이터 시각화 학습 패키지

안녕하세요! "백문이 불여일견"이라는 말이 있듯이, 데이터 분석에서 시각화는 매우 중요합니다. 이번 시간에는 파이썬의 대표적인 시각화 라이브러리, Matplotlib에 대해 알아보겠습니다. Matplotlib를 사용하면 데이터를 다양한 종류의 그래프(플롯)로 그려 패턴, 관계, 분포를 한눈에 파악할 수 있습니다.

※ 참고: Matplotlib는 파이썬 표준 라이브러리가 아니므로, 사용하기 전에 설치해야 합니다.
pip install matplotlib

1. 파이썬 예제 코드: Matplotlib 데이터 시각화

1.1. 기초 파이썬 예시 코드 (5개)

기초 예제는 가장 기본적인 선 그래프를 그리고, 제목과 축 레이블을 추가하는 방법을 다룹니다.

1. 기본적인 선 그래프(Line Plot) 그리기

import matplotlib.pyplot as plt

# x축 데이터와 y축 데이터 준비 (파이썬 리스트)
x = [1, 2, 3, 4]
y = [2, 4, 6, 8]

plt.plot(x, y) # x, y 데이터를 사용하여 선 그래프를 그림
plt.show()     # 그래프를 화면에 표시

2. 제목과 축 레이블 추가하기

import matplotlib.pyplot as plt

# 한글 폰트 설정 (코랩 환경이나 OS에 따라 설정이 다를 수 있음)
plt.rcParams['font.family'] = 'Malgun Gothic' 

x = [1, 2, 3, 4]
y = [10, 20, 25, 30]

plt.plot(x, y)
plt.title("월별 성적 변화") # 그래프 제목
plt.xlabel("월 (Month)")     # x축 이름
plt.ylabel("점수 (Score)")    # y축 이름
plt.show()

3. 선의 스타일과 색상 변경하기

import matplotlib.pyplot as plt

# 'ro--' : 빨간색(r), 점(o) 마커, 점선(--) 스타일
plt.plot([1, 2, 3], [1, 4, 9], 'ro--') 
plt.show()

# 상세 설정을 통한 스타일 변경
plt.plot([1, 2, 3], [1, 2, 3], color='blue', marker='s', linestyle=':', linewidth=2)
plt.show()

4. 여러 개의 그래프 한 번에 그리기 (범례 추가)

import matplotlib.pyplot as plt

days = [1, 2, 3, 4, 5]
temp_hong = [22, 24, 21, 25, 23] # 홍길동의 체온
temp_insu = [20, 22, 25, 22, 21] # 인수샘의 체온

plt.plot(days, temp_hong, label='홍길동')
plt.plot(days, temp_insu, label='인수샘')

plt.legend() # 범례 표시
plt.title("일별 체온 변화 비교")
plt.show()

5. 축 범위 지정하기 (xlim, ylim)

import matplotlib.pyplot as plt

x = [1, 2, 3, 4]
y = [10, 20, 30, 40]

plt.plot(x, y)
plt.xlim(0, 5)  # x축 범위를 0에서 5까지로 설정
plt.ylim(0, 50) # y축 범위를 0에서 50까지로 설정
plt.show()

1.2. 응용 파이썬 예시 코드 (5개)

막대 그래프, 산점도, 히스토그램 등 다양한 형태의 그래프 활용법을 다룹니다.

1. 막대 그래프(Bar Chart)

import matplotlib.pyplot as plt

categories = ['국어', '영어', '수학', '과학']
scores = [85, 90, 78, 92]

plt.bar(categories, scores, color='skyblue')
plt.title("홍길동의 과목별 성적")
plt.show()

2. 산점도(Scatter Plot)

# 두 변수 간의 상관관계를 파악할 때 유용합니다.

import matplotlib.pyplot as plt
import numpy as np

# 무작위 데이터 생성
x = np.random.rand(50)
y = np.random.rand(50)
colors = np.random.rand(50)
area = (30 * np.random.rand(50))**2

plt.scatter(x, y, s=area, c=colors, alpha=0.5) # 크기(s), 색상(c), 투명도(alpha)
plt.title("무작위 분포 산점도")
plt.show()

3. 히스토그램(Histogram)

# 데이터의 빈도 분포를 확인할 때 사용합니다.

import matplotlib.pyplot as plt
import numpy as np

data = np.random.randn(1000) # 표준정규분포를 따르는 난수 1000개

plt.hist(data, bins=30, color='green', edgecolor='black') # 구간 수(bins)
plt.title("데이터 분포 히스토그램")
plt.show()

4. 파이 차트(Pie Chart)

import matplotlib.pyplot as plt

labels = ['사과', '바나나', '딸기', '포도']
sizes = [25, 35, 20, 20]
explode = (0, 0.1, 0, 0) # 바나나 조각만 튀어나오게 설정

plt.pie(sizes, explode=explode, labels=labels, autopct='%1.1f%%', shadow=True)
plt.title("좋아하는 과일 비율")
plt.show()

5. 여러 개의 플롯 나누어 그리기 (Subplots)

import matplotlib.pyplot as plt

x = [1, 2, 3, 4]
y1 = [1, 4, 9, 16]
y2 = [1, 2, 3, 4]

fig, ax = plt.subplots(1, 2) # 1행 2열 구조 생성

ax[0].plot(x, y1, 'r')
ax[0].set_title("제곱")

ax[1].plot(x, y2, 'g')
ax[1].set_title("직선")

plt.tight_layout() # 그래프 간 겹침 방지
plt.show()

1.3. 심화 파이썬 예시 코드 (4개)

Pandas 연동 및 고급 주석 처리, 3D 시각화 기법입니다.

1. Pandas DataFrame과 연동하여 그리기

import matplotlib.pyplot as plt
import pandas as pd

# Pandas DataFrame 생성
data = {'월': [1, 2, 3, 4, 5, 6], '매출': [100, 120, 150, 130, 160, 180]}
df = pd.DataFrame(data)

# DataFrame의 .plot() 메소드를 바로 사용
df.plot(kind='line', x='월', y='매출', marker='o', title='월별 매출 추이')
plt.grid(True) # 그리드(격자) 표시
plt.show()

2. 그래프에 텍스트와 화살표 추가하기 (Annotation)

import matplotlib.pyplot as plt
import numpy as np

x = np.linspace(0, 10, 100)
y = x**2

plt.plot(x, y)
plt.title("Annotation Example")

# 텍스트 추가
plt.text(2, 60, "y = x^2", fontsize=15)

# 화살표와 함께 주석 추가
plt.annotate('최저점', xy=(0, 0), xytext=(2, 20),
             arrowprops=dict(facecolor='black', shrink=0.05))
plt.show()

3. 스타일 시트 적용하기

# 미리 정의된 스타일 테마(ggplot, seaborn 등)를 적용합니다.

import matplotlib.pyplot as plt

plt.style.use('ggplot') # ggplot 스타일 적용

plt.plot([1, 2, 3, 4], [10, 20, 15, 25])
plt.title("ggplot 스타일 적용 결과")
plt.show()

4. 3D 산점도 그리기

import matplotlib.pyplot as plt
import numpy as np

fig = plt.figure()
ax = fig.add_subplot(111, projection='3d') # 3D 축 생성

x = np.random.standard_normal(100)
y = np.random.standard_normal(100)
z = np.random.standard_normal(100)

ax.scatter(x, y, z, c='r', marker='o')
ax.set_title("3D 산점도 (구운몽의 연구 데이터)")
plt.show()
